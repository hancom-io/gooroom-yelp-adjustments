파일 열기
터미널 이용하기

터미널의 명령줄에서 파일 경로를 입력합니다.


$ gnome-mpv /home/.../[파일명]
 
파일 관리자 이용하기

파일 관리자에서 파일을 두 번 클릭하거나 오른쪽 마우스 단추를 클릭한 다음 [동영상으로 열기]를 선택합니다.
        
동영상에서 열기

좌측 상단의 + 아이콘을 클릭하여 [열기]를 통해 동영상 프로그램에서 파일을 열 수 있습니다.
        
드래그 앤 드롭하기

열려는 파일을 동영상 프로그램으로 드래그 앤 드롭합니다.
