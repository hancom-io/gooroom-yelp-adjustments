위치 열기
좌측 상단의 + 아이콘을 클릭하여 [위치 열기]를 통해 동영상 프로그램에서 파일을 열 수 있습니다.
         
[위치 열기]를 클릭하면 나타나는 창의 위치에 열고자 하는 파일명을 포함한 파일의 경로를 입력합니다.

파일의 경로를 입력하는 형식은 다음과 같습니다.

1. 전체 절대 경로 입력


/home/user/비디오/sample.mp4

2. 홈폴더 아래 경로만 입력


비디오/sample.mp4


파일의 경로는 파일 속성에서 확인합니다.
