사용자 계정 삭제 
컴퓨터에 새 사용자 계정을 추가한 후 해당 사용자가 더 이상 컴퓨터를 사용하지 않는 경우 이 계정을 삭제할 수 있습니다.
 
사용자 계정을 삭제하려면 관리자 권한이 필요합니다.
1. [설정] 프로그램을 실행합니다. 아래의 4가지 방법 중 하나를 수행합니다.

- [사적]  아이콘을 클릭한 후 [설정]  아이콘을 클릭합니다.

- 구름 패널 우측의 구름 통합 애플릿에서 [설정]  아이콘을 클릭합니다.

- 
Alt
 + 
F2
를 누르면 나타나는 프로그램 실행 화면에서 'gnome-control-center'를 입력합니다.

- 터미널 창을 열어 'gnome-control-center'를 입력합니다.

2. [세히 보기-사용자]를 선택합니다.

3. 오른쪽 상단에서 [잠금 해제]를 누르고 메시지가 나타나면 암호를 입력합니다.

4. 삭제할 사용자를 선택하고 오른쪽 하단의 [사용자 제거]를 눌러 해당 사용자 계정을 삭제합니다.

5. 각 사용자에게는 파일 및 설정에 대한 고유한 홈 폴더가 있습니다. 사용자의 홈 폴더를 유지하거나 삭제하도록 선택할 수 있습니다. 파일을 더 이상 사용하지 않고 디스크 공간을 확보해야 하는 경우 [파일 삭제]를 누릅니다. 이러한 파일은 영구적으로 삭제되며 복구할 수 없습니다. 파일을 삭제하기 전에 외부 저장 장치에 파일을 백업할 수 있습니다.
