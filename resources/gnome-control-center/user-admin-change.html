관리자 권한이 있는 사용자 변경 
관리자 권한은 시스템의 중요한 부분을 변경해야 하는 사람에게 부여해야 합니다. 특정 사용자를 지정하여 관리자 권한을 부여할 수 있습니다. 관리자 권한은 시스템을 안전하게 보호하고 허용되지 않는 사용자가 시스템을 변경하는 것을 방지할 수 있는 좋은 방법입니다.
 
계정 유형을 변경하려면 관리자 권한이 필요합니다.
1. [설정] 프로그램을 실행합니다. 아래의 4가지 방법 중 하나를 수행합니다.

 - [시작]  아이콘 클릭 후 [설정]  아이콘을 클릭합니다.

- 구름 패널의 구름 통합 애플릿 화면에서 [설정]  아이콘을 클릭합니다.

- 
Alt
 + 
F2
를 누르면 나타나는 프로그램 실행 화면에서 'gnome-control-center'를 입력합니다.

- 터미널 창을 열어 'gnome-control-center'를 입력합니다.

2. [자세히 보기-사용자]를 선택합니다.

3. 오른쪽 상단에서 [잠금 해제]를 누르고 메시지가 나타나면 암호를 입력합니다.

4. 권한을 변경할 사용자를 선택합니다.

5. 계정 종류를 [표준] 또는 [관리자]로 선택합니다.

6. 다음에 로그인할 때 사용자의 권한이 변경됩니다.

시스템의 첫 번째 사용자 계정은 일반적으로 관리자 권한이 있는 계정입니다. 시스템을 처음 설치할 때 생성된 사용자 계정입니다.

한 시스템에 관리자 권한을 가진 사용자가 너무 많은 것은 바람직하지 않습니다.
