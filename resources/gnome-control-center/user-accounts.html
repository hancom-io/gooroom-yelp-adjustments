사용자 계정
컴퓨터를 사용하는 사람마다 다른 사용자 계정이 있어야 합니다. 이를 통해 본인의 파일을 다른 사람의 파일과 구분하고 본인만의 설정을 선택할 수 있으며 보안을 유지할 수 있습니다. 다른 사용계정에 접근하려면 반드시 암호가 필요합니다.

사용자 계정 관리 
새 사용자 계정 추가
       
다른 사람이 컴퓨터에 로그인할 수 있도록 새 사용자를 추가합니다.
 
사용자 계정 삭제
       
더 이상 컴퓨터를 사용하지 않는 사용자 계정을 삭제합니다.
 
로그인 화면 사진 변경
       
로그인 및 사용자 화면에 사진을 추가합니다.
 
자동으로 로그인
       
컴퓨터를 켤 때 자동 로그인되도록 설정합니다.


암호 
암호 변경
       
계정 설정에서 암호를 주기적으로 변경하여 계정을 안전하게 유지합니다.
 
안전한 암호 선택
       
더 길고 복잡한 암호로 변경합니다.


사용자 권한 
관리자 권한이 있는 사용자 변경
       
사용자에게 관리자 권한을 부여하여 시스템을 변경하도록 허용합니다.
 
관리자 제한으로 인한 문제
       
관리자 권한이 있는 경우에만 애플리케이션 설치와 같은 작업을 수행할 수 있습니다.
 
관리자 권한은 어떻게 사용되나요?
       
시스템의 중요한 부분을 변경하려면 관리자 권한이 필요합니다.
