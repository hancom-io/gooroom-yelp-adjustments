유선 네트워크
유선 인터넷 연결

네트워크 케이블이 연결된 경우 유선 네트워크 연결을 설정합니다.
          
VPN 연결

인터넷을 통해 로컬 네트워크로 연결할 수 있도록 VPN을 설정합니다.
          
고정 IP 연결

고정 IP 주소를 사용하여 사용자 컴퓨터에서 보다 쉽게 네트워크 서비스를 제공합니다.
          
수동 네트워크 설정

네트워크 설정이 자동으로 할당되지 않는 경우 사용자가 직접 설정할 수 있습니다.
