배경
사용자의 바탕 화면 및 잠금 화면의 배경을 변경할 수 있습니다. 한컴구름에서 기본적으로 제공되는 이미지 또는 사용자가 추가한 사진으로 배경을 변경할 수 있습니다. 이 설정은 바탕 화면 및 잠금 화면 배경에만 적용되므로 메뉴 또는 아이콘을 변경하려면 테마를 참고하세요.
 

다음 단계를 수행하여 바탕 화면 및 잠금 화면의 배경을 변경할 수 있습니다.
     

1. 아래의 4가지 방법 중 하나를 수행하여 [설정] 애플리케이션을 실행합니다.

- [시작]  아이콘 클릭 후 [설정]  애플리케이션을 선택합니다.

- 구름 패널 오른쪽 하단의 구름 통합 애플릿 화면에서 [설정]  아이콘을 클릭합니다.

- 
Alt
 + 
F2
를 누르면 나타나는 프로그램 실행 화면에서 'gnome-control-center'를 입력합니다.

- 터미널 창을 열어 'gnome-control-center'를 입력합니다.

2. 왼쪽 영역에서 [배경]을 선택합니다.

3. 현재 배경의 상태를 확인합니다.
배경 화면과 잠금 화면을 동일하게 적용하려면 배경 화면 적용 타입에서 [모든 화면 동일 적용]을 선택하고, 배경 화면과 잠금 화면을 별도로 적용하려면 [홈 화면과 잠금 화면 별도 적용]을 선택합니다. 배경 화면을 클릭하면 배경 화면에 체크 박스가 생기고, 잠금 화면을 클릭하면 잠금 화면에 체크 박스가 생깁니다. 체크 박스가 생긴 화면이 사용자가 선택한 이미지/색으로 변경됩니다.

4. 배경 화면과 잠금 화면을 이미지/단일 색으로 설정할 수 있습니다. 이미지로 설정하고 싶은 경우 배경 선택에서 [이미지]를 선택 후 원하는 이미지를 선택하거나, [이미지 추가하기]를 눌러 사용자가 원하는 이미지를 추가하여 선택할 수 있습니다. 단일 색으로 설정하고 싶은 경우 [단일 색상]을 선택 후 원하는 색상을 선택하여 변경할 수 있습니다.


한컴구름 바탕화면의 배경은 [이미지 뷰어] 애플리케이션을 통해서도 변경할 수 있습니다.
