배경
배경 메뉴를 이용하여 사용자가 원하는 배경 화면을 지정할 수 있습니다.
 

배경을 변경하려면 다음의 단계를 수행합니다.
     

1. 아래의 4가지 방법 중 하나를 수행하여 [설정] 애플리케이션을 실행합니다.

- [시작]  아이콘 클릭 후 [설정]  애플리케이션을 선택합니다.

- 구름 패널 오른쪽 하단의 구름 통합 애플릿 화면에서 [설정]  아이콘을 클릭합니다.

- 
Alt
 + 
F2
를 누르면 나타나는 프로그램 실행 화면에서 'gnome-control-center'를 입력합니다.

- 터미널 창을 열어 'gnome-control-center'를 입력합니다.

2. 왼쪽 영역에서 [배경]을 선택합니다.

3. 현재 배경의 상태를 확인합니다.
기본 배경 화면을 변경하려면 [배경]을 클릭하고, 잠금 화면 배경을 변경하려면 [화면 잠그기]를 클릭합니다.

4. 배경 변경 화면의 헤더 바에서 [배경 이미지], [사진], [색]과 같은 탭을 선택하여 원하는 배경으로 변경합니다.



구름 바탕 화면의 배경은 [이미지 뷰어] 애플리케이션을 통해서도 변경할 수 있습니다.
