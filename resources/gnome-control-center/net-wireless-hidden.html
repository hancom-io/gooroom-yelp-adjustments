숨겨진 와이파이에 연결
와이파이를 숨긴 상태로 설정할 수 있습니다. 숨겨진 네트워크는 와이파이 목록에 표시되지 않습니다. 

와이파이를 숨김 설정하려면 다음 단계를 수행합니다.


1. 아래의 4가지 방법 중 하나를 수행하여 [설정] 애플리케이션을 실행합니다.

- [시작]  아이콘 클릭 후 [설정]  애플리케이션을 선택합니다.

- 구름 패널 오른쪽 하단의 구름 통합 애플릿 화면에서 [설정]  아이콘을 클릭합니다.

- 
Alt
 + 
F2
를 누르면 나타나는 프로그램 실행 화면에서 'gnome-control-center'를 입력합니다.

- 터미널 창을 열어 'gnome-control-center'를 입력합니다.

2. [와이파이]을 클릭합니다.

3. 창의 오른쪽 상단에서 [메뉴] 단추을 누르고 [숨겨진 네트워크에 연결…]을 선택합니다.

4. 목록에서 이전에 연결된 숨겨진 네트워크를 선택하거나 [새로 만들기]를 클릭하여 새 네트워크를 추가합니다.

5. 새로 추가하는 경우 네트워크 이름을 입력하고 네트워크 보안 목록에서 무선 보안 유형을 선택합니다.

6. 비밀번호 또는 기타 보안 세부 사항을 입력합니다.

7. [연결]을 클릭합니다.


네트워크 이름을 확인하려면 무선 액세스 포인트 또는 라우터의 설정을 확인해야 할 수 있습니다. 네트워크 이름(SSID)이 없는 경우 BSSID(기본 서비스 세트 식별자, 액세스 포인트의 MAC 주소)를 사용할 수 있습니다. 이는 02:00:01:02:03:04와 유사하며 일반적으로 액세스 포인트 아래에 있습니다.

무선 액세스 포인트에 대한 보안 설정도 확인해야 합니다. WEP 및 WPA와 같은 항목을 찾아보세요.

**와이파이를 숨기면 모르는 사람이 연결하는 것을 방지하여 보안이 향상될 것이라 생각할 수도 있습니다. 하지만 실제로는 그렇지 않으며 네트워크를 찾는 데 시간이 조금 더 걸릴뿐 숨겨진 네트워크를 여전히 감지할 수 있습니다.
