새 사용자 계정 추가 
컴퓨터에 여러 사용자 계정을 추가할 수 있습니다. 회사나 비즈니스와 같은 조직의 모든 구성원에게 하나의 사용자 계정을 발급합니다. 계정이 있는 모든 사용자는 홈 폴더, 문서 및 설정을 소유하게 됩니다.
 
사용자 계정을 추가하려면 관리자 권한이 필요합니다.
1. [설정] 프로그램을 실행합니다. 아래의 4가지 방법 중 하나를 수행합니다.

- [시작]  아이콘 클릭 후 [설정]  애플리케이션을 선택합니다.

- 구름 패널 오른쪽 하단의 구름 통합 애플릿 화면에서 [설정]  아이콘을 클릭합니다.

- 
Alt
 + 
F2
를 누르면 나타나는 프로그램 실행 화면에서 'gnome-control-center'를 입력합니다.

- 터미널 창을 열어 'gnome-control-center'를 입력합니다.

2. [자세히 보기-사용자]를 선택합니다.

3. 오른쪽 상단에서 [잠금 해제]를 누르고 메시지가 나타나면 암호를 입력합니다.

4. 암호를 바르게 입력하면 오른쪽 상단의 단추가 변경됩니다. [사용자 추가(A)...] 단추를 클릭합니다.

5. 새 사용자가 컴퓨터에 대한 관리자 권한을 가지려면 계정 유형에 대한 관리자를 선택합니다.
관리자는 사용자 추가 및 삭제, 소프트웨어 및 드라이버 설치, 날짜 및 시각 변경 등의 작업을 수행할 수 있습니다.

6. 새 사용자의 전체 이름을 입력합니다. 사용자 이름은 전체 이름에 따라 자동으로 입력됩니다. 추천 사용자 이름이 마음에 들지 않으면 변경할 수 있습니다.

7. 새 사용자에 대한 암호를 설정하도록 선택하거나 첫 번째 로그인 시 사용자가 직접 설정하도록 할 수 있습니다. [지금 암호를 설정하도록 선택한 경우] 아이콘을 클릭해 자동으로 임의의 암호를 생성할 수 있습니다.

8. [추가]를 클릭합니다.
계정을 만든 후 암호를 변경하려면 계정을 선택하고 패널을 잠금 해제한 후 현재 암호 상태를 누릅니다.

사용자 패널 오른쪽에서 사용자 이름 옆에 있는 이미지를 클릭하여 계정에 대한 이미지를 추가할 수 있습니다. 이 이미지는 로그인 창에 표시됩니다. 시스템에서 제공하는 사진 또는 사용자의 사진을 사용하거나 웹캠으로 사진을 찍을 수 있습니다.
