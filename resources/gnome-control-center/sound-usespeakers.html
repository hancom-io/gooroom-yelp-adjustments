출력 장치 설정
사용자 컴퓨터에 연결된 소리 출력 장치를 설정합니다. 외부 스피커 혹은 컴퓨터에 내장 스피커가 존재하는 경우 설정이 가능합니다. 소리 출력 장치에 달린 둥근 플러그를 컴퓨터의 스피커 소켓에 연결하면 됩니다. USB 스피커를 사용하는 경우 컴퓨터의 USB 포트에 연결하면 됩니다.

출력 장치는 아래 단계를 진행하여 설정할 수 있습니다.


1. 좌측 하단의 [시작]  아이콘을 클릭하고 검색 창에 "설정"을 입력한 후 [설정] 애플리케이션을 실행합니다.


          

          


2. [설정] 애플리케이션 사이드 바에서 [소리]를 클릭합니다.


          

          


3. [출력] 탭을 클릭하여 연결된 출력 장치를 확인하고, 출력 음량, 좌우 밸런스 및 소리 프로파일을 설정합니다.
