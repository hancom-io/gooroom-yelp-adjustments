접근성


시각 지원
글자 크게 보기 설정

글자 크기를 크게 설정하여 내용을 읽기 쉽게 만듭니다.
          
커서 크기 설정

마우스 포인터를 크기를 크게 변경하여 마우스 위치를 파악하기 쉽게 만듭니다. 
          
키 소리 설정

특정 키를 입력할 때 재생할 소리를 추가합니다.



이동 지원

마우스 움직임
마우스 및 터치패드 속도 조정

마우스 및 터치패드를 움직였을 때 화면에 표시되는 포인터가 움직이는 속도를 설정합니다.
          
마우스 키 설정

키보드 숫자 패드를 사용하여 마우스를 클릭하고 커서를 이동합니다.


클릭과 드래그
마우스 두 번 클릭 속도 설정

사용자가 마우스를 두 번 클릭할 때 첫 번째 클릭과 두 번째 클릭 사이의 속도를 설정합니다.
          
마우스 오른쪽 단추 누르기 대체

마우스 오른쪽 단추를 누르는 대신 마우스 왼쪽 단추를 길게 누르도록 설정합니다.
          
자동 누르기 설정

마우스 자동 누르기를 설정합니다. 마우스 포인터를 가만히 두는 경우 마우스를 클릭한 것으로 인식합니다.


키보드 사용
키보드 탐색

마우스 없이 애플리케이션과 컴퓨터를 사용할 수 있는 키 유형을 제공합니다.
          
반복 키 설정

특정 키를 길게 누르고 있을 때 반복해서 입력되지 않도록 설정합니다. 여기에서는 입력 지연 시간, 속도를 설정할 수 있습니다.
          
탄력 키 사용

동일한 키가 빠르게 여러 번 입력되는 것을 무시합니다. 
          
느린 키 사용

키를 입력한 다음 화면에 글자가 입력되는 시간을 지연시킵니다.
          
고정 키 사용

여러 개의 키를 동시에 누르지 않고 한 번에 하나의 키, 즉 모든 키를 순차적으로 눌러 바로 가기를 실행합니다.
