암호 변경
다른 사람에게 암호가 노출되는 것을 방지하려면 암호를 주기적으로 변경해야 합니다.
 
사용자 계정이 아닌 다른 사용자 계정을 편집하려면 관리자 권한이 필요합니다.
1. [설정] 프로그램을 실행합니다. 아래의 4가지 방법 중 하나를 수행합니다.

- [시작]  아이콘을 클릭한 후 [설정]  아이콘을 클릭합니다.

- 구름 패널 우측의 구름 통합 애플릿에서 [설정]  아이콘을 클릭합니다.

- 
Alt
 + 
F2
를 누르면 나타나는 프로그램 실행 화면에서 'gnome-control-center'를 입력합니다.

- 터미널 창을 열어 'gnome-control-center'를 입력합니다.

2. [자세히 보기-사용자]를 선택합니다.

3. 암호 옆의 ···· 레이블을 클릭합니다. 다른 사용자의 암호를 변경하는 경우 먼저 패널 잠금을 해제해야 합니다.

4. 현재 암호를 입력한 다음 새 암호를 입력합니다. 새 암호 확인 입력란에 새 암호를 다시 입력합니다.

아이콘을 눌러 자동으로 임시 암호를 생성할 수 있습니다.

5. [변경]을 클릭합니다.
 
복잡한 암호를 설정하여 사용자 계정을 안전하게 보호할 수 있습니다.
로그인 암호를 업데이트하면 로그인 키링 암호가 새 로그인 암호와 동일하게 변경됩니다.
암호를 잊어버린 경우 관리자 권한을 가진 사용자가 암호를 변경할 수 있습니다.
