IP 주소란?
IP는 Internet Protocol의 줄임말로 IP 주소는 인터넷 프로토콜 주소를 의미하며 네트워크에 연결된 각 장치는 IP 주소를 가지고 있습니다.

IP 주소는 사용자의 휴대 전화 번호와 비슷합니다. 사용자의 휴대 전화 번호는 고유한 숫자 조합으로 구성되어 있어 사용자를 구별할 수 있기 때문에 다른 사용자에게 전화를 걸 수 있습니다. IP 주소도 이와 유사하게 고유한 숫자 조합으로 구성되어 사용자의 컴퓨터를 구별할 수 있고 데이터를 송신할 수 있습니다.

대부분의 IP 주소는 숫자로 구성된 네 개의 조합으로 구성되며 각 조합은 마침표로 구분됩니다. IP 주소는 192.168.1.42와 같습니다.

IP 주소는 유동적 혹은 고정적일 수 있습니다. 유동 IP 주소는 사용자의 컴퓨터가 네트워크에 연결될 때마다 임시적으로 할당됩니다. 고정 IP 주소는 고정되어 있어 변하지 않습니다. 일반적으로 유동 IP 주소가 사용되며, 고정 IP 주소는 서버 관리자와 같이 특수한 경우에 사용됩니다.
