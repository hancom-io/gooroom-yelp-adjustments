유용한 바로 가기 확인
사용자가 컴퓨터와 애플리케이션을 더욱 효율적으로 사용할 수 있게 해주는 바로 가기를 확인합니다. 만약 마우스나 기타 포인팅 장치가 사용이 불가능한 경우 키보드 탐색을 참고하세요.

- 데스크탑 바로 가기

바로 가기설명
Alt + F2명령어 입력 창을 엽니다. 방향 키를 사용하여 이전에 사용했던 명령어에 빠르게 접근할 수 있습니다.
Alt + Tab현재 창에서 다른 창으로 빠르게 전환합니다. Shift을 눌러 기존 이동 방향과 반대로 이동할 수 있습니다.
Super + O로그아웃 창을 엽니다.
Super + L사용자의 화면을 잠급니다.


- 공통적인 편집 바로 가기

바로 가기설명
Ctrl + A편집 화면의 모든 내용을 선택합니다.
Ctrl + X편집 화면에서 선택한 내용을 잘라냅니다.
Ctrl + C편집 화면에서 선택한 내용을 복사합니다.
Ctrl + V클립보드에 저장된 내용을 편집 화면에 붙여 넣습니다.
Ctrl + Z가장 마지막에 한 행동을 하기 이전으로 되돌립니다.


- 화면 캡쳐

바로 가기동작
Print Screen현재 화면의 스크린샷 이미지를 캡처합니다. 자세한 내용은 화면 캡쳐 를 참고하세요.
Alt + Print Screen현재 창의 스크린샷 이미지를 캡처합니다. 자세한 내용은 화면 캡쳐를 참고하세요.
Shift + Print Screen사용자가 지정한 영역의 스크린샷 이미지를 캡처합니다. 마우스 포인터가 십자 모양으로 바뀌면 마우스 단추를 누른 상태에서 드래그하여 영역을 지정할 수 있습니다. 자세한 내용은 화면 캡쳐를 참조하세요.
