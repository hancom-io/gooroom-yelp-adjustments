관리 제한으로 인한 문제 
관리자 권한이 없는 경우 몇 가지 문제가 발생할 수 있습니다. 작업을 수행하려면 다음과 같은 관리 권한이 필요합니다.

 - 네트워크 또는 와이파이에 연결
 
 - 다른 디스크 파티션(예: Windows 파티션)의 내용 보기
 
 - 새 애플리테이션 설치

관리자 권한을 가진 사용자를 변경할 수 있습니다.
