로그인 화면 사진 변경 
로그인하거나 사용자를 전환하면 로그인 사진과 함께 사용자 목록이 표시됩니다. 기본 사진 또는 본인의 사진으로 변경할 수 있습니다. 웹캠을 사용하여 사진을 바로 찍을 수도 있습니다.
 

다른 사용자 계정을 변경하려면 관리자 권한이 필요합니다.
1. [설정] 프로그램을 실행합니다. 아래의 4가지 방법 중 하나를 수행합니다.

- [시작]  아이콘을 클릭한 후 [설정]  아이콘을 클릭합니다.

- 구름 패널 우측의 구름 통합 애플릿에서 [설정]  아이콘을 클릭합니다.

- 
Alt
 + 
F2
를 누르면 나타나는 프로그램 실행 화면에서 'gnome-control-center'를 입력합니다.

- 터미널 창을 열어 'gnome-control-center'를 입력합니다.

2. [자세히 보기-사용자]를 선택합니다.

3. 다른 사용자 계정을 편집하려면 오른쪽 상단의 [잠금 해제]를 누르고 메시지가 표시되면 암호를 입력합니다.

4. 이름 옆의 사진을 클릭합니다. 기본 사진과 함께 갤러리가 표시됩니다. 원하는 사진을 선택하여 사용합니다.

- 컴퓨터에 저장된 사진을 사용하려면 [파일 선택]을 클릭합니다.

- 웹캠이 있는 경우 [사진 찍기]를 클릭하여 새 사진을 찍습니다. 사진의 크기를 조정하고 불필요한 영역을 잘라냅니다.
