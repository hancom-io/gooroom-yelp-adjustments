블루투스 장치에 파일 보내기

휴대폰 또는 다른 컴퓨터와 같은 블루투스 장치에 파일을 보낼 수 있습니다. 일부 장치에서는 파일 전송이 허용되지 않거나 특정 유형의 파일이 전송되지 않습니다. 블루투스 설정 창에서 파일을 전송할 수 있습니다.

※ iPhone과 같은 지원되지 않는 장치에서는 파일을 보낼 수 없습니다.        
      



1. [블루투스]를 클릭하여 패널을 엽니다.

2. 블루투스가 활성화되어 있는지 확인합니다. 제목 표시줄의 스위치가 켜져 있어야 합니다.

3. 장치 목록에서 파일을 보낼 장치를 선택합니다. 원하는 장치가 목록에서 '연결됨'으로 표시되어 있지 않으면 연결해야 합니다.

4. [파일 보내기…]를 클릭하면 파일 선택 화면이 나타납니다.

5. 원하는 파일을 선택하고 [선택]을 클릭합니다.
   하나의 폴더에서 두 개 이상의 파일을 보내려면 각 파일을 선택할 때마다 Ctrl을 길게 누릅니다.

6. 일반적으로 수신자는 버튼을 눌러야 파일을 받을 수 있습니다. [블루투스 파일 전송] 대화 상자에 진행률 표시줄이 표시됩니다. 전송이 완료되면 [닫기]를 클릭합니다.
