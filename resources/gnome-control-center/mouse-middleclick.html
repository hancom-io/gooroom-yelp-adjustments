마우스 중간 클릭
대부분의 마우스 및 터치패드 장치에는 마우스 중간 단추가 포함되어 있습니다. 스크롤 휠이 있는 마우스에서 스크롤 휠을 직접 누르면 중간 클릭이 실행됩니다. 마우스에 중간 버튼이 없는 경우 마우스 왼쪽 및 오른쪽 단추를 동시에 누르면 중간 클릭으로 동작합니다.

여러 개의 손가락 탭을 지원하는 터치패드에서는 세 손가락을 사용하여 동시에 탭하여 중간 클릭을 실행할 수 있습니다. 보다 자세한 내용은 터치패드 클릭, 드래그, 스크롤 설정을 참고하세요.

많은 애플리케이션에서 클릭 바로 가기 설정을 위해 중간 클릭을 사용합니다.

스크롤 막대의 빈 공간에서 마우스 왼쪽 단추를 누르면 스크롤이 클릭한 위치로 바로 이동합니다. 중간 클릭의 경우 현재 페이지보다 한 페이지 위로 올라갑니다.

대부분의 웹 브라우저에서 링크를 새 탭으로 열 때 마우스 중간 클릭을 사용합니다. 링크에서 마우스 중간 단추를 누르면 해당 링크가 새 탭으로 열립니다.

[파일]에서 중간 클릭은 두 가지 역할을 합니다. 첫째, 폴더 이름 위에서 마우스 중간 단추를 누르면 해당 폴더가 새 탭으로 열립니다. 이는 웹 브라우저의 기능과 동일합니다. 둘째, 파일 이름 위에서 마우스 중간 단추를 누르면 해당 파일이 엽립니다. 이는 파일을 두 번 클릭한 것과 동일합니다.

일부 애플리케이션에서 마우스 중간 단추를 누르면 특수한 기능이 실행되며 이에 대한 자세한 내용은 해당 애플리케이션 도움말을 참고하시기 바랍니다.
