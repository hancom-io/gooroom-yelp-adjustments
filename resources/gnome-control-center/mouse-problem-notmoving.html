마우스 포인터가 움직이지 않음 
- 마우스가 제대로 연결되었는지 확인

- 마우스가 고장나지 않았는지 확인

- 무선 마우스인지 확인



마우스 연결 상태 확인
유선 마우스를 사용하는 경우 컴퓨터에 올바르게 연결되었는지 확인합니다. USB를 사용하는 마우스인 경우 컴퓨터의 다른 USB 포트에 연결해봅니다. PS/2 마우스인 경우, 보라색 키보드 포트가 아닌 초록색 마우스 포트에 연결되었는지 확인합니다. 연결 확인을 위해 컴퓨터를 재시작해야 할 수 있습니다.



마우스 고장 여부 확인
연결하려는 마우스를 다른 컴퓨터에 연결하여 해당 마우스가 정상적으로 동작하는지 확인합니다. 옵티컬 또는 레이저 마우스를 사용하는 경우 마우스가 켜져 있으면 마우스 바닥에서 빛이 나옵니다. 빛이 나오지 않으면 마우스가 켜져 있는지 확인합니다. 마우스가 켜져 있음에도 빛이 나오지 않는다면 마우스가 고장난 것일 수 있습니다.



무선 마우스 사용 여부 확인
마우스가 켜져 있는지 확인합니다. 마우스 바닥에서 스위치를 확인할 수 있습니다.

블루투스 마우스를 사용하는 경우 마우스가 컴퓨터와 제대로 페어링되어 있는지 확인합니다. 자세한 내용은 컴퓨터에 블루투스 장치 연결 페이지를 참고하세요.

마우스 포인터가 움직인다면 마우스 단추를 눌러 봅니다. 일부 무선 마우스에서는 절전 모드로 인해 단추를 누르기 전까지 작동하지 않을 수 있습니다. 자세한 내용은 마우스 동작이 지연됨 페이지를 참고하세요.

무선 마우스 배터리가 충전되어 있는지 확인하세요.

컴퓨터 블루투스 수신 장치가 정상적으로 연결되었는지 확인하세요.

마우스와 수신기가 다른 채널에서 동작한다면 두 채널을 동일하게 맞춰주세요.

마우스와 수신기를 설정하기 위해 마우스와 수신기에 있는 단추를 동시에 눌러보세요. 이에 대한 자세한 내용은 마우스 매뉴얼을 확인해 보시기 바랍니다.

대부분의 RF 무선 마우스는 컴퓨터에 연결되면 자동으로 동작합니다. 블루투스 혹은 IR 무선 마우스를 사용하기 위해 사전 설정이 필요할 수 있습니다. 설정 방법은 사용 중인 마우스의 모델에 따라 다릅니다.
