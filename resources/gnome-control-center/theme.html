테마
한컴구름은 Windows, macOS, Android와 같은 다른 운영 체제를 이미 경험해본 많은 사용자들이 익숙하고 친근하게 사용할 수 있도록 설계되었습니다. 시각적 효과를 높이고 더 나은 사용성을 제공하기 위해 한컴구름에서는 [테마]를 제공합니다.


1. 아래의 4가지 방법 중 하나를 수행하여 [설정] 애플리케이션을 실행합니다.

- [시작]  아이콘 클릭 후 [설정]  애플리케이션을 선택합니다.

- 구름 패널 오른쪽 하단의 구름 통합 애플릿 화면에서 [설정]  아이콘을 클릭합니다.

- 
Alt
 + 
F2
를 누르면 나타나는 프로그램 실행 화면에서 'gnome-control-center'를 입력합니다.

- 터미널 창을 열어 'gnome-control-center'를 입력합니다.

2. 왼쪽 메뉴에서 [테마]를 선택합니다. 

3. [현재 테마]에서 현재 적용되어 있는 테마를 확인합니다.

4. [테마 목록]에서 원하는 테마를 선택하고 자신만의 스타일로 한컴구름을 꾸밉니다.
