자동으로 로그인
컴퓨터를 시작할 때 계정에 자동으로 로그인되도록 설정을 변경할 수 있습니다.
1. [설정] 프로그램을 실행합니다. 아래의 4가지 방법 중 하나를 수행합니다.

- [시작]  아이콘을 클릭한 후 [설정]  아이콘을 클릭합니다.

- 구름 패널 우측의 구름 통합 애플릿에서 [설정]  아이콘을 클릭합니다.

- 
Alt
 + 
F2
를 누르면 나타나는 프로그램 실행 화면에서 'gnome-control-center'를 입력합니다.

- 터미널 창을 열어 'gnome-control-center'를 입력합니다.

2. [자세히 보기-사용자]를 선택합니다.

3. 시작할 때 자동으로 로그인할 사용자 계정을 선택합니다.

4. 오른쪽 상단에서 [잠금 해제]를 누르고 메시지가 나타나면 암호를 입력합니다.

5. 자동 로그인 스위치를 켭니다.
 
다음에 컴퓨터를 시작하면 자동으로 로그인됩니다. 이 옵션을 사용하도록 설정한 경우 계정에 로그인하기 위해 암호를 입력할 필요가 없습니다. 즉, 다른 사용자가 컴퓨터를 시작하면 계정과 파일 및 브라우저 기록을 포함한 개인 데이터에 액세스할 수 있습니다.
계정 유형이 표준인 경우 이 설정을 변경할 수 없습니다. 이 설정을 변경할 수 있는 시스템 관리자에게 문의해야 합니다.
