키보드 탐색
마우스 또는 마우스와 유사한 장치를 사용할 수 없거나, 키보드를 많이 사용하는 사용자에게 유용한 키를 설명합니다. 모든 사용자에게 유용한 바로 가기 키를 확인하려면 유용한 바로 가기 키 확인 페이지를 참고하세요.

마우스와 같은 장치를 사용할 수 없는 경우 키보드의 숫자 키패드를 사용하여 마우스 포인터를 제어할 수 있습니다. 이에 대한 자세한 내용은 키패드를 사용하여 마우스 포인터 이동 페이지를 참고하세요.

- 사용자 인터페이스 안내

바로 가기 키설명
Tab 및

Ctrl + Tab		키보드의 포커스를 다른 곳으로 옮깁니다. Ctrl + Tab 키는 사이드 바에서 메인 콘텐츠로 이동과 같이 컨트롤 그룹 간의 이동을 담당합니다. Tab 키를 이용하여 컨트롤 그룹 간 이동에서 벗어날 수 있습니다. Shift을 눌러 반대 방향으로 이동할 수 있습니다.
방향 키하나의 컨트롤 또는 연관된 컨트롤 내에서 이동을 담당합니다. 방향 키를 사용하여 도구 모음에서 버튼을 강조 표시하거나 목록 또는 아이콘 보기에서 항목을 선택하거나 그룹에서 라디오 버튼을 선택할 수 있습니다.
Ctrl + 방향 키목록 또는 아이콘 보기에서 키보드 포커스를 다른 항목으로 옮깁니다.
Shift + 방향 키목록 또는 아이콘 보기에서 현재 선택된 항목부터 새롭게 강조 표시된 항목까지 모든 항목을 선택합니다. 트리 구조에서 자식 노드를 확장시키거나 축소시킬 때 사용됩니다. 자식 노드 확장을 위해서는 Shift + →, 축소를 위해서는 Shift + ← 를 입력합니다.
Space버튼, 확인란, 목록과 같이 선택된 항목을 실행합니다.
Ctrl + Space목록 또는 아이콘 보기에서 다른 항목에 영향을 주지 않고 강조 표시된 항목을 선택하거나 선택 해제합니다.
Alt메뉴 항목, 버튼 및 기타 컨트롤에서 바로 가기 키로 사용할 수 있는 문자에 밑줄이 표시됩니다. Alt 키와 밑줄 친 문자를 함께 누르면 해당 컨트롤이 실행되며 마우스로 클릭한 것과 동일한 결과가 나타납니다.
Esc메뉴, 팝업, 전환기, 대화 상자 등을 종료합니다.
F10현재 창에 있는 메뉴 모음에서 첫 번째 메뉴를 엽니다. 방향 키를 사용하여 메뉴를 이동할 수 있습니다.
Shift + F10 또는

Menu		선택된 항목의 컨텍스트 메뉴를 표시합니다. 해당 항목에서 마우스 오른쪽 단추를 눌렀을 때와 동일하게 동작합니다.
Ctrl + F10파일 관리자에서 현재 폴더의 컨텍스트 메뉴를 표시합니다. 어떤 항목도 선택하지 않고 마우스 오른쪽 단추를 눌렀을 때와 동일하게 동작합니다.
Ctrl + PageUp
및
Ctrl + PageDown		여러 개의 탭으로 구분된 인터페이스에서 PageUp 키를 누르면 왼쪽 탭으로, PageDown 키를 누르면 오른쪽 탭으로 이동합니다.


- 데스크톱 및 창 안내

바로 가기 키동작
Alt + Tab현재 창에서 다른 창으로 빠르게 전환합니다. Shift 키를 눌러 반대 방향으로 이동할 수 있습니다.
Alt + F4현재 창을 닫습니다.
Super + ↑현재 창을 최대화합니다. 최대화된 상태에서 다시 누르면 최대화 이전으로 되돌아갑니다.
Super + ←현재 창을 모니터 좌측 절반만큼 최대화합니다. 좌측 절반만큼 최대화된 상태에서 다시 누르면 최대화 이전으로 되돌아갑니다.
Super + →현재 창을 모니터 우측 절반만큼 최대화합니다. 우측 절반만큼 최대화된 상태에서 다시 누르면 최대화 이전으로 되돌아갑니다.
Super + H현재 창의 가로를 최대화합니다. 창의 가로가 최대화된 상태에서 다시 누르면 최대화 이전으로 되돌아갑니다.
Super + V현재 창의 세로를 최대화합니다. 창의 세로가 최대화된 상태에서 다시 누르면 최대화 이전으로 되돌아갑니다.
Alt + Space창 메뉴를 표시합니다. 현재 창의 제목 표시줄에서 마우스 오른쪽 단추를 누른 것과 동일하게 동작합니다.
