기본식
기본식 표준 수학 형태로 입력합니다. 예를 들어 7과 2를 더하려면 다음과 같이 입력합니다.


7+2


값을 구하려면 마우스로 = 단추를 누르거나 키보드의 Enter 키를 누릅니다.
모든 기본식은 사칙연산의 순서를 따릅니다. 덧셈과 뺄셈보다 곱셈과 나눗셈을 우선 처리합니다. 다음 수식의 결과 값은 1입니다. (3×2 = 6, 7−6 = 1)


7−3×2


계산 순서를 바꾸려면 괄호를 사용합니다. 다음 수식의 결과 값은 8입니다. (7−3 = 4, 4×2 = 8)


(7−3)×2


표시 값을 지우려면 Ctrl 키 또는 Escape 키를 누릅니다.
