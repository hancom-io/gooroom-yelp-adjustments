구름 OS 설치
다음의 순서에 따라 구름 OS를 설치하실 수 있습니다.


실행하기
부트 로더에서 원하는 항목을 선택한 후 Enter를 클릭하면 라이브 모드로 부팅됩니다. 아래의 과정에 따라 '구름 설치'를 실행합니다.



1.런처  버튼을 선택한 다음 작업 표시줄의 검색 상자에 '구름 설치 '을 입력하고 결과 목록에서 구름 설치를  선택합니다.
        

2.런처  버튼을 선택한 다음 오른쪽 프로그램 모음에서 구름 설치  아이콘을 선택합니다.


설치하기
다음 8가지 순서에 따라 구름OS를 설치합니다.
1. 언어



[언어] 페이지에서 Korean을 선택하고 '앞으로'를 누릅니다.


2. 시간대



[시간대] 페이지에서는 지역과 시간대를 선택할 수 있습니다. 지도에서 Asia/Seosul을 선택하고 '앞으로'를 누릅니다.


3. 키보드 배치



[키보드 배치] 페이지에서 Korean을 선택하고 '앞으로'를 누릅니다. 키보드 구성은 대부분 자동으로 인식되면 아래 입력란에서 테스트할 수 있습니다.


4. 사용자 정보 



[사용자 정보] 페이지에서 필요한 항목을 적절히 입력합니다. 특히 입력한 사용자 이름과 암호를 분실하지 않도록 주의합니다.


5. 파티션 설정 
[파티션 설정] 페이지는 환경에 따라 정보가 달리 표시될 수 있습니다.



이전에 OS를 설치한 적 없는 새 디스크인 경우 '예'를  선택하여 새 파티션을 생성합니다




기본값은  /boot/efi가 100MB 이상(fat32), /(루트) 30GB (ext4) 로 설정되어 있습니다. 더 상세한 편집을 위해서 파티션 편집을 참고하세요.


6. 고급 옵션 



[고급 옵션] 페이지에서 부트로더와 암호화 방식을 선택합니다. 기본값은 Ecryptfs입니다.


7. 개요



[개요] 페이지에서는 현재까지의 선택값을  한 눈에 확인할 수 있습니다. 모든 사항이 맞으면 설치버튼을 클릭하여 설치를 진행합니다.


8. 구름플랫폼 설치중...



설치가 완료되면 [설치완료] 안내창이 뜹니다.  '예'를 눌러 컴퓨터를 재부팅합니다.


파티션 편집
구름OS를 설치하면서 파티션을 설정할 수 있습니다. 아래의 순서를 따라 편집하세요.. 

1. 파티션 편집 창 열기



[파티션 설정] 페이지 오른쪽 하단의 파티션 편집 버튼을 클릭하여 편집창을 엽니다. 새로운 파티션을 추가하기 위해서는 왼쪽 상단의 +모양의 아이콘을 클릭해주세요. 또는 마우스 우클릭하여 새로 만들기 메뉴를 선택합니다.


2. EFI 파티션 만들기



첫 번째 파티션은 부팅을 위한 EFI 파티션으로 설정해 줍니다.  크기는 100MB이상, 파일 시스템은 fat32로 입력합니다. 입력이 완료된 후에 추가 버튼을 누릅니다.


3. EFI 플래그 설정



만들어진 파티션을 선택한 후 마우스 우클릭하여 플래그 관리 메뉴를 선택합니다.



boot 체크박스를 선택한 후 닫기 버튼을 클릭합니다.


4. root 파티션 만들기



두 번째 파티션은 구름 사용을 위한 /(root) 파티션으로 지정합니다. 새 크기는 30GB 이상, 파일 시스템은 ext4로 선택합니다. 입력이 완료된 후에 추가 버튼을 누릅니다.

5. root 플래그 설정
EFI 플래그 설정과 마찬가지로, 마우스 우클릭하여 플래그 관리 메뉴를 선택 후 /(root)로 플래그를 지정합니다. 


6. 파티션 저장



모든 편집이 끝나면 상단의 ✔ 버튼을 클릭합니다. 경고메시지가 나오면 '적용'을 누릅니다.


7. 적용



변경사항을 적용하기 위해 새로 고침  을 클릭하고, 구름을 설치할 파티션을 클릭합니다. 마우스 우클릭 후 '/로 할당합니다'  메뉴를 선택합니다. /파티션을 지정한 후 다음 단계로 넘어갑니다.
