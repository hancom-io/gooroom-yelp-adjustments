웹 스토어
구름 브라우저는 구글 웹 스토어에서 제공하는 앱과 호환됩니다.

직접 구글 웹 스토어에 접근하거나, 다음 방법을 사용하여 구글이 제공하는 웹 스토어에 접근할 수 있습니다.


구름 브라우저 주소창에 gooroom://extensions 를 타이핑하고 엔터를 누릅니다.


            



위 페이지에서 좌측 상단의 메뉴 아이콘을 클릭하고 [Gooroom browser 호환 웹 스토어 열기]를 클릭합니다.

            


연결된 구글 웹 스토어에서 원하는 확장 프로그램을 찾을 수 있습니다.

            


원하는 확장 프로그램을 선택하고 설치합니다.

            


설치된 익스텐션을 해당 페이지나, 우측 상단의 아이콘을 우클릭하여 삭제할 수 있습니다.
