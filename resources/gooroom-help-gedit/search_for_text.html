텍스트 검색
찾기 도구를 사용하면 파일에서 특정 텍스트 시퀀스를 찾을 수 있습니다.


텍스트 찾기 
1. 메뉴 버튼 ▸ 찾기를 클릭하거나 Ctrl + F를 눌러 검색 창을 엽니다.
         
2. 검색 창에 검색할 텍스트를 입력합니다.
 
3. 입력한 내용과 일치하는 텍스트 영역이 강조 표시됩니다.


검색 결과를 스크롤하려면 다음 중 하나를 수행합니다.
- 검색 창 옆에 있는 위쪽 또는 아래쪽 화살표를 클릭합니다.

- 키보드에서 위쪽 화살표 또는 아래쪽 화살표 키를 누릅니다.

Ctrl + G 또는 Ctrl + Shift + G를 누릅니다.


검색 창을 닫으려면 Esc 또는 Enter를 누릅니다. Esc를 누르면 커서가 검색을 시작하기 전 위치로 돌아갑니다. Enter를 누르면 커서가 검색 결과의 현재 위치로 돌아갑니다.


검색 팁 
- 마우스로 텍스트를 강조 표시한 다음 Ctrl + F를 누르면 강조 표시한 텍스트가 검색 창에 나타납니다.


- 자세한 검색 옵션을 보려면 검색 창에서 돋보기 아이콘을 클릭하거나 검색 창의 아무 곳이나 마우스 오른쪽 단추로 클릭합니다. 다음 검색 옵션 중 하나 이상을 선택할 수 있습니다.


대소문자를 구분하여 검색하려면 [대/소문자 구분]을 선택합니다.

전체 단어만 검색하려면 [전체 단어만 일치]를 선택합니다.

텍스트를 위에서 아래로 검색한 다음 처음부터 다시 검색하려면 [둘러쓰기]를 선택합니다.


- 검색한 텍스트는 검색을 완료한 후에도 강조 표시됩니다. 강조 표시를 제거하려면 [강조 표시 지우기]를 클릭하거나 Ctrl + Shift + K를 누릅니다.
