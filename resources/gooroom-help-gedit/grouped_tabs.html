그룹화된 탭으로 파일 구성
여러 탭으로 작업하는 경우 해당 탭을 그룹화할 수 있으므로 열려 있는 파일을 쉽게 정리할 수 있습니다. 새 탭 그룹을 추가하면 텍스트 편집기 창이 두 개의 창으로 분할되고 새로운 "제목 없는 문서"가 새 창에 열립니다. 파일을 해당 탭 그룹으로 열고 탭을 다른 탭 그룹으로 이동할 수 있습니다.

새 탭 그룹 열기
새 탭 그룹을 열려면 다음을 수행합니다.


    - 메인 메뉴에서 Documents ▸ New Tab Group(문서 ▸ 새 탭 그룹)으로 이동합니다.

    - 또는 Ctrl + Alt + N을 누릅니다.

이렇게 하면 텍스트 편집기 창이 두 개의 창으로 나뉘고 새 탭 그룹이 있는 창이 활성 탭 오른쪽에 배치됩니다. 필요에 따라 핸들을 오른쪽 또는 왼쪽으로 이동하여 텍스트 편집기 창 공간을 적절히 할당할 수 있습니다.

텍스트 편집기 창에서 화면이 허용하는 한 여러 탭 그룹을 열 수 있지만 핸들을 올바르게 보려면 핸들을 이동해야 합니다.

탭 그룹 창을 닫으려면 그룹 내에서 열려 있는 모든 탭을 닫습니다.
