탭 이동 및 순서 변경
텍스트 편집기의 탭을 이동하여 파일을 쉽게 사용할 수 있습니다. 탭을 동일한 창 내에서 다시 정렬하거나 다른 창으로 이동할 수 있습니다.


텍스트 편집기 창에서 탭 순서 변경 
창에서 탭 순서를 변경하려면 다음을 수행합니다.
1. 탭에서 마우스 왼쪽 버튼을 길게 누릅니다.

2. 탭을 원하는 위치로 이동합니다.

3. 마우스 왼쪽 버튼을 놓습니다.
탭을 해제하는 경우에는 열려 있는 다른 탭 바로 옆에 배치됩니다.


탭 이동하여 새 창 만들기 
기존 탭에서 새 창을 만들려면 다음을 수행합니다.
1. 탭에서 마우스 왼쪽 버튼을 길게 누릅니다.

2. 텍스트 편집기 창에서 탭을 드래그합니다.

3. 마우스 왼쪽 버튼을 놓습니다.
최대화할 때 탭을 새 창으로 이동하려면 화면 상단의 막대로 드래그합니다.


탭을 다른 창으로 이동 
다른 창으로 탭을 이동하려면 다음을 수행합니다.
1. 탭에서 마우스 왼쪽 버튼을 길게 누릅니다.

2. 탭을 새 창의 다른 탭으로 드래그합니다.

3. 마우스 왼쪽 버튼을 놓습니다.
탭을 GNOME Shell의 Activity 핫코너로 드래그하면 다른 창으로 더 쉽게 이동할 수 있습니다. 이렇게 하면 열려 있는 각 텍스트 편집기 창이 표시됩니다. 그런 다음 원하는 텍스트 편집기 창에서 탭을 해제할 수 있습니다.
 
탭 관련 바로 가기 키 표를 참조하여 탭을 쉽게 관리할 수 있습니다.
