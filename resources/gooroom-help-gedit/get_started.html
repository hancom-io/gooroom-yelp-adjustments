텍스트 편집기 시작하기
텍스트 편집기를 사용하여 간단한 메모 및 문서를 작성하거나 고급 기능을 일부 활용하여 나만의 소프트웨어 개발 환경을 만들 수 있습니다. 

텍스트 편집기를 실행하는 즉시 쓰기를 시작할 수 있습니다. 텍스트를 저장하려면 헤더 바의 도구 모음에서 [저장]을 클릭합니다.

추가 기능을 살펴보거나 다른 작업을 수행하는 방법을 알아보려면 도움말의 다른 섹션을 참조합니다.
