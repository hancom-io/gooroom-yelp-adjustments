서버의 파일 열기 

텍스트 편집기에서 서버의 파일을 열려면 해당 서버의 정보를 알고 있어야 합니다. 예를 들어 서버의 IP 주소, URL 및 서버 유형(예: HTTP 또는 FTP 등) 정보가 필요합니다.

또한 일부 서버 유형에서는 저장된 파일을 사용하여 수행할 수 있는 작업이 일부 제한될 수 있습니다(예: 읽기 전용. 변경 내용은 로컬 디스크에만 저장할 수 있음).

서버에서 파일을 열려면 다음을 수행합니다.
1. 제목 표시줄에서 [열기]를 클릭합니다.

2. [기타 문서...]를 클릭합니다.

3. [열기] 대화 상자에서 Ctrl+L을 누릅니다.

4. 해당 서버의 IP 주소 또는 URL을 입력합니다.



5. 열려는 파일을 선택합니다.

6. [문자 인코딩] 드롭다운 목록을 사용하여 적절한 문자 인코딩을 선택합니다.

7. [열기]를 누릅니다.


올바른 URI 유형에는 http:, ftp:, ftp:, file: 및 gvfs에서 지원되는 모든 유형이 포함됩니다.
일부 URI 유형에서는 파일이 읽기 전용으로 열리며 변경 내용은 다른 위치에 저장해야 합니다. 예를 들어 HTTP는 파일 읽기만 허용합니다. 
         
일부 FTP 서버의 경우 원격 파일 저장이 올바르게 작동하지 않을 수 있으므로 FTP의 파일은 읽기 전용으로 열립니다.
