전체 화면 모드 사용
텍스트 편집기에서 전체 화면 모드로 작업하는 것이 도움이 될 수 있습니다. 전체 화면 모드를 사용하면 메뉴 모음, 탭 모음 및 도구 모음이 숨겨져 더 많은 텍스트가 표시되며 작업에 더욱 집중할 수 있습니다.

전체 화면 모드 켜기 
전체 화면 모드를 켜려면 F11 을 누릅니다. 텍스트 편집기의 메뉴, 제목 및 탭 모음이 숨겨지고 현재 파일의 텍스트만 표시됩니다.

전체 화면 모드에서 작업하는 동안 텍스트 편집기 메뉴에서 작업을 수행해야 하는 경우 마우스 포인터를 화면 상단으로 이동합니다. 텍스트 편집기 메뉴 모음이 다시 나타나고 원하는 작업을 선택할 수 있습니다.


전체 화면 모드에서 열려 있는 탭 사이를 전환하려면 Ctrl + Alt + PgUp 또는 Ctrl + Alt + PgDn을 누릅니다.


전체 화면 모드 끄기 
전체 화면 모드를 끄고 일반 모드로 돌아가려면 F11을 누릅니다.

마우스 커서를 화면 상단으로 이동하고 메뉴 모음이 나타날 때까지 기다릴 수도 있습니다. 메뉴 모음이 나타나면 [전체 화면으로 이동] 버튼을 선택합니다.
