색 고르기
색 고르기 플러그인은 특정 색상을 나타내는 문자 코드를 삽입하는 데 도움이 됩니다. 이렇게 하면 프로그램에서 올바른 색상을 쉽게 선택하고 사용할 수 있습니다.


색상 선택기 플러그인을 사용하도록 설정하려면 메뉴 ▸ 기본설정의 플러그인 탭 ▸ 색 고르기를 선택합니다.

색 고르기를 사용하려면 다음을 수행합니다.


1. 색 코드를 삽입할 문서의 지점에 커서를 놓습니다.

2. [메뉴] ▸ [색 편집]을 선택합니다. [색상 선택] 대화 상자가 나타납니다.

3. 색상 선택기 창에서 원하는 색상을 클릭합니다.

4. 커서를 놓았던 곳에 색상 코드가 삽입됩니다.

5. [닫기]를 클릭하여 색 선택기를 닫습니다.


Hue, Pato, Value, Red, Green 또는 Blue의 값을 조정하여 색상을 미세 조정할 수 있습니다.

또한 색 고르기에서 색상 상자를 마우스 오른쪽 버튼으로 클릭하고 [여기서 색상 저장]을 선택하여 색상을 저장할 수 있습니다. 이렇게 하면 자주 사용하는 색상을 나중에 쉽게 삽입할 수 있습니다.
