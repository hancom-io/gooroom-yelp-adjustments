Python 콘솔
아래쪽 창에 Python 콘솔을 추가하여 Python 스크립트를 테스트할 수 있습니다. Python 콘솔을 사용하도록 설정하려면 헤더 바 메뉴 ▸ 기본 설정 ▸ 플러그인 ▸ Python Console을 선택합니다.

Python 콘솔이 활성화되면 View ▸ Bottom Panel을 선택하여 콘솔을 열 수 있습니다.


내장 터미널 플러그인도 사용하도록 설정한 경우 아래쪽 창에 Python Console이 별도 탭으로 나타납니다.
