파일 열기 

파일을 텍스트 편집기로 열려면 [열기]를 클릭하거나 Ctrl + O를 누릅니다.

이렇게 하면 [열기] 대화 상자가 나타납니다. 마우스 또는 키보드를 사용하여 열려는 파일을 선택한 다음 [열기]를 누르면 선택한 파일이 새 탭에서 열립니다.

파일을 열지 않고 대화 상자를 닫으려면 [취소]를 클릭합니다.
Ctrl 및 Shift 키를 사용하여 여러 파일을 한 번에 열 수 있습니다. 여러 파일을 선택하는 동안 Ctrl 키를 누른 상태에서 [열기]를 클릭하면 선택한 각 파일이 열립니다.

여러 파일을 선택하는 동안 Shift를 누르고 있으면 첫 번째 파일과 마지막 파일을 포함하여 선택한 모든 파일이 열립니다.
