맞춤법 검사
맞춤법 검사 기능은 필요에 따라 사용할 수 있는 gedit의 플러그인으로 제공됩니다. 플러그인을 사용하도록 설정하려면 다음을 수행합니다.


1.gedit ▸ Preferences ▸ Plugins를 선택합니다.

2.플러그인을 사용하려면 철자 검사기를 선택합니다.


텍스트 편집기에서 맞춤법 검사기 사용하기
문서에서 철자가 틀린 단어를 확인하려면 다음을 수행합니다.
도구 > 철자 검사를 선택합니다.

입력할 때 텍스트 편집기에서 문서에서 철자가 틀린 단어를 자동으로 강조 표시하려면 도구 > 철자가 틀린 단어 강조 표시를 선택합니다.


맞춤법 검사기 사용하기
철자 검사 대화 상자를 사용하여 철자가 틀린 단어에 대한 올바른 대체 방법을 선택할 수 있습니다. 철자 검사기에서 확인 중인 철자가 틀린 단어 옆에 굵게 표시됩니다. 다음 수정 작업 중에서 선택할 수 있습니다.


-추천 단어를 사용하면 단어에서 사용할 수 있는 대체 단어를 나열합니다.

- 무시를 사용하면 현재 단어 인스턴스에 대한 맞춤법 검사를 무시할 수 있습니다.

- 모두 무시를 사용하면 현재 워드의 모든 인스턴스에 대한 맞춤법 검사를 무시할 수 있습니다.

- 바꾸기는 잘못된 단어나 인식되지 않는 단어를 제안: 목록에서 선택한 대체 단어로 대체합니다.

- 모두 바꾸기 목록에서 선택한 대체 단어 또는 인식되지 않는 모든 대체 단어를 변경합니다.

- 단어 추가를 사용하여 현재 단어를 gedit 사용자 사전에 추가할 수 있습니다. 사용자 사전에 사용자 지정 단어를 추가하면 gedit가 문서의 단어를 인식할 수 있으므로, 철자가 틀린 단어로 강조 표시되지 않습니다.


맞춤법 검사기 정의
기본적으로 gedit는 현재 언어를 사용하여 문서의 철자를 검사합니다. 다른 언어를 사용해야 하는 경우 도구 ▸ 언어 설정을 선택한 다음 사용할 언어를 선택합니다.


사전
gedit는 철자 검사를 위해 시스템 유틸리티인 Enchant를 사용합니다. Enchant는 여러 사전을 사용하여 맞춤법을 검사할 수 있습니다(예: Hunspell 및 Aspell 등) .
    
사용할 언어를 gedit에서 사용할 수 없는 경우 컴퓨터의 소프트웨어 설치 관리자 또는 패키지 관리자를 사용하여 원하는 사전을 설치합니다.
