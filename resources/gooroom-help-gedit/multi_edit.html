Multi Edit Mode(다중 편집 모드)
다중 편집 모드 플러그인을 사용하면 여러 위치에서 동시에 문서를 편집할 수 있습니다. 이 기능은 지루한 편집 작업을 수행할 때 유용합니다. 다중 편집 플러그인을 사용하도록 설정하려면 메뉴 ▸ 기본설정 ▸ 플러그인 탭 ▸ 다중 편집을 선택합니다.


플러그인을 사용하도록 설정하면 Ctrl + Shift + C를 누르거나 gedit ▸ Multi Edit Mode를 클릭하여 Multi Edit 모드를 켭니다. 그런 다음 파일에서 편집을 시작할 지점을 선택할 수 있습니다.

파일에 편집 지점을 삽입하려면 다음을 수행합니다.


1. 편집을 시작할 위치에 커서를 둡니다.

2. Ctrl + E를 누릅니다.

3. 다음 편집 지점으로 이동한 다음 Ctrl + E를 다시 누릅니다.

4. 필요에 따라 편집 지점을 계속해서 추가합니다.


편집 지점을 모두 삽입했으면 원하는 내용을 편집할 수 있습니다.


편집 지점을 제거하려면 Esc를 누릅니다. 이렇게 하면 모든 편집 지점이 제거됩니다. Esc를 다시 누르면 다중 편집 모드가 해제되고 일반 편집 모드로 돌아갑니다.
