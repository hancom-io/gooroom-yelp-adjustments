빠르게 열기
빠르게 열기 플러그인을 사용하면 일반적인 파일 파일 열기보다 더 빨리 파일을 열 수 있습니다. 이 플러그인을 사용하도록 설정하려면 헤더 바의 메뉴 ▸ 기본 설정 ▸ 플러그인 ▸ 빠르게 열기를 선택합니다.

빠르게 열기 플러그인을 사용하도록 설정하면 다음과 같이 플러그인을 사용할 수 있습니다.
         

1. Ctrl + Alt + O를 누르거나 [빠르게 열기]를 선택하여 [빠르게 열기] 대화 상자를 엽니다.

2. 마우스를 사용하여 열려는 파일을 선택합니다.


여러 파일을 선택하려면 Ctrl 키를 누른 상태에서 열려는 파일을 클릭합니다.

3. 파일 그룹을 선택하려면 열려는 첫 번째 파일을 클릭하고 Shift 키를 누른 상태에서 마지막 파일을 클릭하고 [열기]를 클릭합니다.


빠르게 열기는 다음 위치에서 파일을 찾습니다.


- 현재 열려 있는 문서의 디렉터리

- 파일 브라우저 창 플러그인의 루트 디렉터리

- 최근에 사용한 파일

- 파일에서 책갈피로 작성한 디렉터리

- 바탕 화면 디렉터리

- 홈 디렉터리
