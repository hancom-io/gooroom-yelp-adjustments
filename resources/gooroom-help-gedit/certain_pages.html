특정 페이지 인쇄
문서에서 특정 페이지만 선택적으로 인쇄할 수 있습니다. 예를 들어 페이지 번호 "1,3,5-7,9"를 입력하면 1,3,5,6,7,9페이지만 인쇄됩니다. 


원하는 페이지를 인쇄하려면 다음을 수행합니다.

         
1. File ▸ Print...(파일 ▸ 인쇄...)를 선택합니다.

2. [인쇄] 대화 상자의 [일반] 탭 [범위] 항목에서 페이지를 선택합니다.

3. 문서에서 인쇄할 페이지의 번호를 쉼표로 구분하여 입력합니다. 대시보드를 사용하여 페이지 범위를 지정합니다.
