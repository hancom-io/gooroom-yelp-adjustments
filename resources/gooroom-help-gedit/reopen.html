최근에 사용한 파일 다시 열기 

텍스트 편집기에서는 기본적으로 가장 최근에 사용한 파일 5개에 쉽게 액세스할 수 있습니다. 최근에 사용한 파일을 열려면 다음을 수행합니다.

1. [열기]를 누릅니다.

2. 최근에 사용한 파일 5개가 목록으로 나열됩니다.

3. 원하는 파일을 선택하면 새 탭에서 해당 파일이 열립니다.
dconf-editor 애플리케이션을 사용하여 다음과 같이 최근에 사용한 파일의 표시 개수를 조정할 수 있습니다.

1. dconf-editor를 시작하고 org ▸ gnome ▸ gedit ▸ preferences ▸ ui rec max-resents를 선택합니다.

2. 최대 사용 횟수 옆의 숫자를 두 번 누르고 원하는 값으로 변경합니다.
