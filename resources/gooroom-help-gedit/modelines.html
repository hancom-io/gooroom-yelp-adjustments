모드라인
모드라인 플러그인을 사용하면 gedit에서 파일의 시작과 끝에 있는 텍스트 행을 분석하여 환경 설정을 일괄 적용할 수 있습니다. 모드라인 플러그인은 Emacs, Kate 및 Vim 텍스트 편집기에서 사용되는 옵션의 하위 집합을 지원합니다. 모드라인 플러그인을 사용하도록 설정하려면 텍스트 편집기 ▸ 상단 버튼 ▸ 기본설정 ▸ 플러그인 ▸ 모드라인를 선택합니다.

일반 옵션
하단의 모드라인을 사용하여 다음 옵션을 설정할 수 있습니다.
         

- 탭 너비

- 들여쓰기 너비

- 탭 대신 공백 삽입

- 텍스트 줄 바꿈

- 오른쪽 여백 너비


**모드라인 설정은 [기본 설정] 대화 상자 설정보다 우선합니다.


Emacs 모드라인
Emacs 모드라인는 문서의 처음 두 행을 스캔합니다. gedit에서는 다음 Emacs 모드라인 옵션을 지원합니다.


- 탭 너비

- 들여쓰기 오프셋

- Indent-tabs-mode

- 텍스트 자동 줄 바꿈


Emacs 모드라인에 대한 자세한 내용은 GNU Emacs Manual을 참조합니다.


Kate 모드라인
Kate 모드라인는 문서의 첫 번째 행과 마지막 행을 스캔합니다. gedit에서는 다음 Kate 모드라인 옵션을 지원합니다.

          

- 탭 너비

- 들여쓰기 너비

- space-indent

- word-wrap

- word-wrap-column


Kate 모드라인에 대한 자세한 내용은 Kate Website를 참조합니다.


Vim 모드라인
Vim 모드라인는 문서의 처음 행 3개와 마지막 행 3개를 스캔합니다. gedit에서는 다음 Vim 모드라인 옵션을 지원합니다.
		 
		 

- et (expandtab)

- ts(tabstop)

- sw(shiftwidth)

- wrap

- textwidth


Vim 모드라인에 대한 자세한 내용은 Vim Website를 참조합니다.
