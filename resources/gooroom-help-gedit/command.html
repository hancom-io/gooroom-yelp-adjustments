명령어
명령어 플러그인은 현재 파일에서 명령줄 작업을 수행할 수 있는 고급 사용자를 위한 플러그인입니다. 명령어 플러그인을 사용하도록 설정하려면 메뉴 ▸ 기본 설정 ▸ 플러그인 탭 ▸ 명령어를 선택합니다.


플러그인을 사용하도록 설정한 다음 Ctrl + .(마침표)를 눌러 플러그인을 활성화합니다.
동작바로 가기 키
사용 가능한 명령 개요 보기Tab + Tab
현재 파일에서 단어 찾기/foo
단어 foo를 찾는 명령어입니다.
현재 파일 내에서 단어 바꾸기/ + /foo bar
이렇게 하면 foo가 bar로 바뀝니다.
커서를 파일의 특정 줄 번호로 이동go [라인 번호] 또는
G + [라인 번호]
현재 줄의 책갈피 표시기 전환bookmark 혹은 B 
bookmark 명령과 같은 일부 명령에서는 해당하는 플러그인을 설치하고 활성화해야 합니다. 플러그인이 설치되지 않았거나 활성화되지 않은 경우 오류 메시지가 표시됩니다.
