올바른 순서로 인쇄할 복사본 가져오기
인쇄 순서 반전 
프린터는 일반적으로 첫 페이지를 먼저 인쇄하고 마지막 페이지를 마지막으로 인쇄합니다. 인쇄 순서를 반대로 하려면 다음을 수행합니다.

         
1.파일 ▸ 인쇄를 선택합니다.

2. [인쇄] 대화 상자의 [일반] 탭에서 [반전]을 선택합니다.


이렇게 하면 마지막 페이지가 먼저 인쇄됩니다.


한 부씩 인쇄 
문서의 복사본을 두 개 이상 인쇄하는 경우 인쇄물은 기본적으로 페이지 번호별로 그룹화됩니다. 데이터 정렬을 수행하면 각 사본에 해당 페이지가 함께 그룹화됩니다.
       

1. File ▸ Print(파일 ▸ 인쇄)를 클릭합니다.

2. [인쇄] 대화 상자의 [일반] 탭에서 [복사본]을 선택합니다.
