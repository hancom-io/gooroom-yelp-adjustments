텍스트 바꾸기
텍스트 편집에 시간이 오래 걸릴 수 있습니다. 텍스트 편집기에는 편집 시간을 절약하기 위해 일부 텍스트를 찾아 바꾸는 데 도움을 주는 찾아서 바꾸기 기능이 포함되어 있습니다.


텍스트 편집기에서 텍스트 바꾸기 
1. 메뉴에서 찾기 및 바꾸기를 클릭하여 바꾸기 도구를 열거나 Ctrl + H를 누릅니다.
         
2. [찾기] 필드에 찾으려는 텍스트를 입력합니다.
 
3. [바꾸기] 필드에 바꾸려는 텍스트를 입력합니다.

4. 각 필드에 텍스트를 입력한 후 검색 조건에 매개 변수를 추가할 수 있습니다. 또는 바꿀 항목을 선택할 수도 있습니다.
    - 다음 일치 항목만 바꾸려면 [바꾸기]를 클릭합니다.
    - 찾은 텍스트를 모두 바꾸려면 [모두 바꾸기]를 클릭합니다.
[모두 바꾸기]는 텍스트 파일 전체에 대해 작동하며 바꾸기를 수행할 텍스트 영역을 강조 표시하지 않으므로 주의해야 합니다.


추가 옵션 
원하는 경우 검색 조건에 다음 매개 변수를 추가할 수 있습니다.
대소문자 일치(M)
대/소문자 구분 옵션을 사용하여 대/소문자를 구분할지 여부를 지정할 수 있습니다. 이 옵션을 선택하면 대소문자를 구분하여 검색합니다.
 
전체가 맞는 단어만(E)
다른 단어의 일부를 포함하지 않고 특정 단어를 검색하려면 이 옵션을 사용합니다. 예를 들어 이 옵션을 선택한 상태에서 'and'를 검색하면 단어 'and'는 검색되지만 'sand' 및 'commander'와 같은 단어는 검색되지 않습니다.

뒤로 검색(B)
[이전 찾기] 명령과 동일하게 작동합니다. 검색 결과를 처음부터 끝까지 진행하려면 이 옵션을 선택합니다.
  
끝까지 가면 처음부터(P)
줄바꿈 옵션이 활성화된 상태에서 파일의 마지막까지 검색한 다음 파일 처음에서 검색/바꾸기 작업을 다시 시작합니다. 이렇게 하면 전체 파일에 걸쳐 검색/바꾸기 작업이 수행됩니다.  

정규식(G)
정규식을 이용하여 문자열을 찾고 변경할 수 있습니다. 예를 들어, 메모장에 아래와 같은 내용이 입력되어 있습니다.
  one.tim
  one.ben
  one.greg

찾기(i)에 ([a-z]+)\.([a-z]+) 를 입력하고, 바꿀 문자열(W)에 \2/\1 를 입력한 후 모두 바꾸기(A)를 클릭하면
다음과 같은 결과를 출력합니다.
  time/one
  ben/one
  greg/one


이스케이프 시퀀스를 사용 
이스케이프 시퀀스를 사용하여 줄 바꿈, 캐리지 리턴 및 탭을 검색할 수 있습니다.
\nNewline
\rCarriage return
\tTab 
텍스트에서 \(백슬래시) 또는 이스케이프 시퀀스 중 하나를 검색하려면 추가 백슬래시를 사용하여 백슬래시를 이스케이프 처리해야 합니다. 예를 들어 \를 검색하려면 [찾기] 또는 [바꾸기] 필드에 \\를 입력해야 합니다. 이를 \n으로 바꾸려면 \\\n를 사용해야 합니다. 같은 방법으로 이스케이프 시퀀스를 검색할 수 있습니다.
