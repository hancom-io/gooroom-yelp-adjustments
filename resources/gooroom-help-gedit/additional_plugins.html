추가 플러그인
타사 플러그인을 설치하면 기본 gedit 플러그인에서 사용할 수 없는 추가 기능 또는 gedit-plugins 패키지의 일부인 추가 기능을 사용할 수 있습니다.


타사 플러그인을 설치하려면 플러그인을 다운로드하여 올바른 디렉터리에 배치한 다음 활성화해야 합니다.


1. 일반적으로 플러그인 웹 사이트에서 플러그인을 다운로드할 수 있지만 인터넷에서 플러그인의 소프트웨어 리포지토리를 복사하려면 git, bazaar 또는 subversion과 같은 수정 제어 프로그램을 사용해야 할 수 있습니다. 플러그인 가져오기에 대한 자세한 내용은 해당 플러그인의 설명서를 참조합니다.

2. 플러그인 파일을 /home/username/.local/share/gedit/plugins 디렉터리에 배치합니다(username은 실제 사용자 이름으로 변경).


3. 플러그인 파일을 올바른 디렉터리에 배치하면 플러그인이 기본 설정 메뉴에 나타납니다. 플러그인을 사용하도록 설정하려면 Edit preferences Preferences ▸ Plugins ▸ [Name of Plugin]을 선택합니다.

4. Close를 클릭합니다.



.local 디렉터리는 기본적으로 숨겨집니다. 파일 및 기타 숨김 파일을 보려면 [보기] ▸ [숨김 파일 표시]를 선택하거나 Ctrl+H를 누릅니다.

.local/share/gedit/plugins/ 디렉터리가 시스템에 없는 경우 해당 디렉터리를 생성합니다.
