루트 사용자로 파일 편집 
루트 사용자로 파일을 편집하는 작업은 잠재적으로 위험하며 이로 인해 시스템이 손상될 수 있습니다. 파일을 루트 사용자로 편집할 때는 특히 주의해야 합니다.

루트 사용자로 파일을 편집하려면 다음을 입력하여 터미널에서 텍스트 편집기를 시작합니다.
sudo gedit
sudo 명령을 사용하여 gedit를 열려면 먼저 암호를 입력해야 합니다.
         
sudo 명령 대신 다음을 입력하여 관리자 권한을 얻을 수 있습니다.
su -
그런 다음 gedit 명령을 사용하여 텍스트 편집기를 시작합니다.
         
관리자 권한으로 텍스트 편집기를 열면 닫기 전까지 해당 권한이 유지됩니다.
