파일 닫기 

텍스트 편집기에서 파일을 닫으려면 Close(닫기)를 선택합니다. 또는 파일 탭 오른쪽에 있는 "X"를 클릭하거나 Ctrl + W를 누릅니다.

파일에 저장되지 않은 변경 사항이 있는 경우 파일을 닫기 전에 저장할지 묻는 메시지가 표시됩니다.
