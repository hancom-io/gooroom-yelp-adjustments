코드 주석
코드 주석은 프로그램의 코드를 쉽게 이해할 수 있도록 설명하는 텍스트입니다. 프로그램의 작동에는 영향을 주지 않습니다.
코드 주석은 특수 문자로 구분됩니다.


코드 주석 플러그인을 사용하면 코드 주석을 더 쉽게 만들거나 제거할 수 있습니다. 코드 주석 플러그인을 사용하려면 메뉴 ▸ 기본설정 ▸ 플러그인 탭 ▸ 코드 주석을 선택합니다.

주석을 작성하거나 삭제하려면 다음을 수행합니다.


1. 설명을 추가하거나 삭제할 텍스트 영역을 강조 표시합니다.

2. 설명을 추가하려면 gedit ▸ Comment Code를 클릭하거나 Ctrl + M을 누릅니다.

3. 주석을 제거하려면 gedit ▸ Uncontent Code를 클릭하거나 Shift + Ctrl + M을 누릅니다.


편집 지점을 모두 삽입했으면 원하는 내용을 편집할 수 있습니다.


편집 지점을 제거하려면 Esc를 누릅니다. 이렇게 하면 모든 편집 지점이 제거됩니다. Esc를 다시 누르면 다중 편집 모드가 해제되고 일반 편집 모드로 돌아갑니다.
