시작하기
다음 순서에 따라 매체제어 편집기를 시작해주세요.
        
1.런처  버튼을 선택한 다음 작업 표시줄의 검색 상자에 매체제어 편집기를 입력하고 결과 목록에서 매체제어 편집기를 선택합니다.

2.런처  버튼을 선택한 다음 오른쪽 프로그램 모음에서 매체제어 편집기  아이콘을 선택합니다.

3. 사용자 암호를 입력하면 매체제어 편집기가 실행됩니다.


메뉴 설명
왼쪽 상단 메뉴 버튼을 클릭하면 다음과 같은 기능들이 존재합니다.

다시읽기(R)로컬에 저장되어 있는 매체제어 제어 상태를 불러옵니다.
저장(S)매체제어 편집기에서 편집한 상태를 저장합니다.
서버정책삭제(U)GPMS계정으로 로그인 시,
서버에서 내려주는 정책을 삭제할 수 있습니다.
도움말매체제어 편집기의 도움말을 보여줍니다.
