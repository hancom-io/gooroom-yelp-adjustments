일반적으로 사용되는 문서 유형에 대한 템플릿

파일 템플릿을 사용하면 유사한 서식의 문서를 쉽게 생성할 수 있습니다.
 


새 템플릿 만들기
1. 템플릿으로 사용할 문서를 만듭니다.

2. 홈 폴더의 [서식] 폴더에 템플릿 콘텐츠로 파일을 저장합니다. 서식 폴더가 없다면 폴더를 먼저 생성해야 합니다.

템플릿을 사용하여 문서 만들기
1. 새 문서를 저장할 폴더를 엽니다.

2. 폴더의 빈 영역을 마우스 오른쪽 버튼으로 클릭한 다음 [새 문서(D)]를 선택합니다. 사용 가능한 템플릿이 나열됩니다.

3. 목록에서 원하는 템플릿을 선택합니다.

4. 파일을 두 번 클릭하여 열고 편집합니다. 편집을 완료하면 파일 이름을 변경하여 저장합니다.
