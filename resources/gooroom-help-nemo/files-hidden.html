파일 숨기기

[파일 관리자]은 숨기기 및 숨기기 취소 기능을 제공합니다. 숨김 파일은 [파일 관리자]에서 표시되지 않지만 삭제된 것이 아닙니다.

파일을 숨기려면 파일 이름이 . 접두사로 시작하도록 변경합니다. 예를 들어 example.txt라는 파일을 숨기려면 파일 이름을 .example.txt로 변경합니다.

폴더 이름에 . 접두사를 추가하면 폴더를 숨길 수도 있습니다.


숨김 파일 모두 표시
폴더 내의 모든 숨김 파일을 표시하려면 해당 폴더로 이동한 다음 도구 모음에서 [보기 옵션]을 클릭하고 [숨김 파일 표시]를 선택하거나 Ctrl+H를 누릅니다. 이렇게 하면 숨김 파일을 포함한 모든 파일이 표시됩니다.

숨김 파일을 다시 표시하지 않으려면 [보기 옵션]을 클릭하고 [숨김 파일 표시]를 선택 해제하거나 Ctrl+H를 다시 누릅니다.



파일 숨기기 해제
파일 숨기기를 해제하려면 숨김 파일이 포함된 폴더로 이동한 다음 도구 모음에서 [보기 옵션]을 클릭하고 [숨김 파일 보이기(H)]를 선택합니다. 
그런 다음 숨김 파일 이름에서 . 접두사를 삭제합니다. 예를 들어 .example.txt 라는 파일을 숨김 해제하려면 파일 이름을 example.txt로 변경합니다.

도구 모음에서 [보기 옵션]을 클릭하고 [숨김 파일 표시]를 선택 해제하거나 Ctrl+H를 눌러 숨김 파일을 다시 숨길 수 있습니다.

숨김 파일 이름에는 ~ 접미사가 있을 수 있습니다. 이러한 파일은 백업 파일입니다. 자세한 내용은 접미사 ~가 포함된 파일이 무엇입니까?를 참조합니다.
