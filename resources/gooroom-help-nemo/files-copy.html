파일 및 폴더 복사/이동

마우스로 드래그 앤 드롭하거나 복사 및 붙여넣기 명령을 사용하거나 키보드 바로 가기 키를 사용하여 파일 또는 폴더를 새 위치로 복사하거나 이동할 수 있습니다.

예를 들어 프레젠테이션을 메모리 카드에 복사하여 함께 작업하거나 문서를 변경하기 전에 원본 문서의 백업 사본을 생성할 수 있습니다.

이 절차는 파일과 폴더에 적용됩니다. 동일한 방법으로 파일 및 폴더를 복사하거나 이동할 수 있습니다.


파일 복사 및 가져오기
1. 복사하려는 파일을 한 번 클릭하여 선택합니다.

2. 마우스 오른쪽 버튼을 클릭하고 [복사]를 선택하거나 Ctrl+C를 누릅니다.

3. 파일 사본을 저장할 폴더로 이동합니다.

4. 메뉴 버튼을 클릭하고 [붙여넣기]를 선택하여 파일을 복사하거나 Ctrl+V를 누릅니다.



잘라내기 후 붙여넣기로 파일 이동
1. 이동하려는 파일을 한 번 클릭하여 선택합니다.

2. 마우스 오른쪽 버튼을 클릭하고 [잘라내기]를 선택하거나 Ctrl+X를 누릅니다.

3. 파일을 이동할 폴더로 이동합니다.

4. 메뉴 버튼을 클릭하고 [붙여넣기]를 선택하여 파일을 이동하거나 Ctrl+V를 누릅니다.



드래그 앤 드롭으로 파일 복사 또는 이동
1. 파일 관리자를 열고 복사하려는 파일이 있는 폴더로 이동합니다.

2. 상단 표시줄에서 파일을 클릭하고 새 창을 선택하거나 Ctrl+N를 눌러 두 번째 창을 연 다음 새 창에서 파일을 이동하거나 복사할 폴더로 이동합니다.

3. 파일을 마우스 왼쪽 버튼으로 길게 눌러 다른 창으로 드래그한 다음 버튼을 뗍니다. 이렇게 하면 대상 위치가 동일한 장치인 경우 파일이 이동되고 다른 장치인 경우 복사됩니다.

   - 예를 들어 USB 메모리에서 홈 폴더로 파일을 드래그하면 대상 위치가 서로 다른 장치이므로 파일이 복사됩니다.

   - Ctrl을 누른 상태에서 드래그하여 파일을 강제로 복사하거나 Shift를 누른 상태에서 드래그하여 파일을 강제로 이동할 수 있습니다.



주의 사항
일부 읽기 전용 폴더로는 파일을 복사하거나 이동할 수 없습니다. 파일 권한을 변경하여 읽기 전용 항목을 변경할 수 있습니다.
