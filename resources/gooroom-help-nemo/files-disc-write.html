CD 또는 DVD에 파일 쓰기

Brasero와 같은 디스크 굽기 애플리케이션을 사용하여 빈 디스크에 파일을 쓸 수 있습니다.


디스크가 올바르게 생성되지 않는 경우
데이터가 디스크에 올바르게 기록되지 않거나 디스크의 물리적인 손상으로 인해 파일을 읽지 못하는 경우가 있습니다.

이 경우 디스크 굽기를 다시 시도하되 더 낮은 굽기 속도(예: 48배속 대신 12배속)를 사용합니다. 일반적으로 느린 속도를 사용하는 경우에 데이터를 더 안정적으로 기록할 수 있습니다.
