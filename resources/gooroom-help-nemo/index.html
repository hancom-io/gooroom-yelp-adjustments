파일 관리자 
파일 관리자는 간단하고 통합된 방법으로 파일과 어플리케이션을 관리할 수 있게 합니다.

파일 관리자는 파일을 폴더로 정리할 수 있게 도와줍니다. 폴더는 파일 뿐 아니라 다른 폴더도 포함할 수 있습니다.

폴더를 이용하면 파일을 더 쉽게 관리할 수 있습니다.




일반적인 작업
파일 및 폴더 찾아보기

파일 관리자로 파일을 관리하고 구성합니다.
 
파일 및 폴더 복사/이동

항목을 새 폴더로 복사하거나 이동합니다.
 
파일 및 폴더 삭제

더 이상 필요하지 않은 파일 또는 폴더를 삭제합니다.
 
파일 및 폴더 이름 변경하기

파일 또는 폴더의 이름을 변경합니다.
 
파일 검색

이름 또는 파일 유형을 기준으로 파일을 검색합니다.
 
파일 및 폴더 정렬

이름, 크기 또는 유형 등을 기준으로 파일을 정렬합니다.





더 많은 파일 관련 작업
서버 또는 네트워크 공유에서 파일 찾아보기

FTP, SSH, Windows 공유 또는 WebDAV를 통해 다른 컴퓨터에서 파일을 보고 편집합니다.
 
파일 관리자 환경 설정

파일 브라우저 기본 설정을 확인하고 변경합니다.
 
파일 속성

파일 기본 정보를 확인하고 권한을 설정하거나 기본 애플리케이션을 지정합니다.
 
파일 찾기

파일 검색 방법을 알아봅니다.
 
다른 애플리케이션으로 파일 열기

특정 파일 유형을 열도록 설정된 기본 애플리케이션을 변경하거나 다른 애플리케이션으로 엽니다.
 
휴지통에서 파일 복원

삭제한 파일을 원래 위치로 복원합니다.
 
CD 또는 DVD에 파일 쓰기

CD/DVD 라이터를 사용하여 빈 CD 또는 DVD에 데이터를 기록합니다.





이동식 드라이브 및 외장 디스크
안전하게 외장 드라이브 제거

USB 플래시 드라이브, CD, DVD 또는 기타 외부 장치를 안전하게 연결 해제합니다.

유용한 정보
폴더 책갈피 편집

파일 관리자에서 책갈피를 추가 또는 삭제하거나 이름을 바꿉니다.
 
파일 숨기기

파일을 표시하지 않도록 설정합니다.
 
패턴으로 파일 선택

파일 권한 설정

일반적으로 사용되는 문서 유형에 대한 템플릿

사용자 지정 파일 템플릿을 사용하여 새 문서를 빠르게 생성합니다.
