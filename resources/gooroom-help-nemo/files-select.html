패턴으로 파일 선택

파일 이름의 패턴을 사용하여 파일을 선택할 수 있습니다.
 


Ctrl+S를 눌러 일치하는 항목 선택 창을 불러옵니다. 파일 이름의 공통 부분과 와일드 카드 문자를 사용하여 패턴을 입력합니다. 
다음 와일드 카드 문자를 사용할 수 있습니다.

 - *는 일치하는 모든 문자를 의미합니다. 모든 숫자 및 문자가 해당됩니다.

 - ?는 일치하는 문자 하나를 의미합니다.



예를 들어 다음 패턴을 사용할 수 있습니다.

 - 파일 이름이 Invoice인 모든 파일 유형을 선택하려면 아래와 같이 입력합니다.


Invoice.*


- 파일 이름이 Vacation-001.jpg, Vacation-002.jpg, Vacation-003.jpg와 같은 파일을 선택하려면 아래와 같이 입력합니다.


Vacation-???.jpg


- 위 파일 이름 패턴에 'revised' 접미사가 포함된 파일을 선택하려면 아래와 같이 입력합니다.


Vacation-???-revised.jpg
