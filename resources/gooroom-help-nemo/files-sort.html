파일 및 폴더 정렬

파일을 날짜 또는 크기순 등 다양한 방법으로 정렬할 수 있습니다. 파일을 정렬하는 일반적인 방법은 파일 정렬 방법을 참조합니다.
기본 정렬 순서를 변경하는 방법은 파일 보기 환경 설정을 참조합니다.

파일 정렬 방법은 사용 중인 폴더 보기 설정에 따라 다릅니다. 메뉴의 [보기(V)] 또는 아이콘 버튼을 사용하여 현재 보기를 변경할 수 있습니다.


아이콘 보기
파일 정렬 방법을 변경하려면 메뉴의 [보기(V)] 또는 마우스 오른쪽 버튼을 누르면 나타나는 메뉴에서 [내용 정렬 방식(G)]을 클릭한 다음 목록에서 원하는 항목을 선택합니다.

예를 들어 [이름]을 선택하면 파일이 알파벳순으로 정렬됩니다. 다른 옵션은 파일 정렬 방법을 참조합니다.



목록 보기
파일 정렬 방법을 변경하려면 파일 관리자에서 열 제목 중 하나를 클릭합니다. 예를 들어 [유형]을 클릭하여 파일 유형별로 정렬합니다. 열 제목을 한번 더 클릭하면 역순으로 정렬됩니다.

더 많은 속성 열을 표시하여 해당 열을 기준으로 정렬할 수도 있습니다. 열 영역에서 마우스 오른쪽 버튼을 클릭한 다음 표시할 열을 선택합니다. 사용 가능한 열에 대한 설명은 파일 목록 열 환경 설정을 참조합니다.



촘촘히 보기
파일 정렬 방법을 변경하려면 파일 관리자에서 열 제목 중 하나를 클릭합니다. 예를 들어 [유형]을 클릭하여 파일 유형별로 정렬합니다. 열 제목을 한번 더 클릭하면 역순으로 정렬됩니다.
  
더 많은 속성 열을 표시하고 해당 열을 기준으로 정렬할 수도 있습니다. 도구 모음에서 [표시할 열(V)…] 을 클릭한 다음 표시할 열을 선택합니다. 사용 가능한 열에 대한 설명은 파일 목록 열 환경 설정을 참조합니다.
파일 정렬 방법
이름
파일 이름을 기준으로 알파벳순으로 정렬합니다.

크기
파일 크기(디스크가 차지하는 공간)를 기준으로 정렬합니다. 기본적으로 파일 크기가 작은 항목부터 오름차순으로 정렬합니다.

유형
파일 유형을 기준으로 알파벳순으로 정렬합니다. 같은 유형의 파일은 함께 그룹화되고 이름별로 정렬합니다.

마지막 수정
파일이 마지막으로 변경된 날짜 및 시간을 기준으로 정렬합니다. 기본적으로 오래된 항목부터 최신 항목 순서로 정렬합니다.
