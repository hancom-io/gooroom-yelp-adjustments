파일 및 폴더 찾아보기

[파일 관리자]로 컴퓨터에서 파일을 찾고 구성합니다. 

[파일 관리자]를 사용하면 저장 장치(예: 외장 하드 디스크), 파일 서버 및 네트워크 공유에 있는 파일을 관리할 수도 있습니다.

[파일 관리자]를 시작하려면 활동 개요에서 파일을 엽니다. 애플리케이션 검색 방법과 동일하게 개요를 통해 파일과 폴더를 검색할 수도 있습니다.


폴더 내용 탐색
파일에서 폴더를 두 번 클릭하여 엽니다. 파일을 두 번 클릭하거나 마우스 가운데 버튼 클릭하여 기본 애플리케이션으로 엽니다.
폴더를 마우스 가운데 버튼으로 클릭하면 새 탭에서 엽니다. 폴더를 마우스 오른쪽 버튼으로 클릭하여 새 탭이나 새 창에서 열 수도 있습니다.


- 파일 및 폴더 목록 위의 경로 표시줄에는 현재 폴더가 상위 폴더와 함께 표시됩니다.
- 경로 표시줄에서 상위 폴더를 클릭하여 해당 폴더로 이동합니다. 
- 경로 표시줄에서 폴더를 마우스 오른쪽 버튼으로 클릭하여 새 탭 또는 새 창에서 열거나 속성에 액세스합니다.
- 현재 폴더에서 파일을 빠르게 검색하려면 파일 이름을 입력합니다. 
- 검색 창은 창의 상단 또는 하단에 나타나고 입력과 일치하는 파일만 표시됩니다. 


사이드 바에서 일반적인 위치에 빠르게 액세스할 수 있습니다. 
사이드 바가 표시되지 않으면 창의 오른쪽 상단에 있는 메뉴 버튼을 누른 다음 [사이드 바]를 선택합니다. 
자주 사용하는 폴더에 책갈피를 추가하면 사이드 바에 표시됩니다. 
폴더를 책갈피로 추가하려면 사이드 바로 드래그하거나 [책갈피(B)-책갈피 추가(A)]를 클릭합니다.
