파일 목록 열 환경 설정

파일 목록 보기에서 표시할 수 있는 정보 열은 14개입니다. 창의 오른쪽 상단에 있는 메뉴 버튼을 클릭하고 [기본 설정]을 선택한 다음 [목록 열] 탭을 선택하여 표시할 열을 선택합니다.

[기본값으로 재설정]을 클릭하면 기본 열 표시로 돌아갑니다.
 


이름
폴더 및 파일의 이름입니다. ※이름 열은 숨길 수 없습니다.


크기
폴더 크기는 폴더에 포함된 항목 개수로 제공됩니다. 파일 크기는 Byte, KB 또는 MB로 표시됩니다.


형식
PDF 문서, JPEG 이미지, MP3 오디오 등과 같은 폴더 또는 파일 형식으로 표시됩니다.


수정한 날짜
파일이 마지막으로 수정된 날짜를 제공합니다.


자세한 형식
자세한 형식을 표시합니다.


접근한 날짜
파일이 마지막으로 열린 날짜 또는 해당 시간을 제공합니다.


소유자
폴더 또는 파일을 소유한 사용자 이름을 제공합니다.


그룹
파일을 소유한 그룹을 표시합니다. 각 사용자는 일반적으로 자신의 그룹에 속하지만 하나의 그룹에 여러 사용자가 있을 수 있습니다. 예를 들어 팀 그룹에는 여러 팀 구성원이 포함될 수 있습니다.


권한
파일 액세스 권한을 표시합니다(예: drwxrw-r--)

 - 첫 번째 문자는 파일 유형입니다. -는 일반 파일을 의미하고 d는 디렉터리(폴더)를 의미합니다.

 - 다음 문자 3개(rwx)는 파일을 소유한 사용자의 권한을 지정합니다.

 - 그 다음 문자 3개(rw-)는 파일을 소유한 그룹의 모든 구성원에 대한 권한을 지정합니다.

 - 마지막 문자 3개(r--)는 시스템의 다른 모든 사용자에 대한 권한을 지정합니다.

각 권한에는 다음과 같은 의미가 있습니다.

 - r: 읽기 가능. 파일 또는 폴더를 열 수 있습니다.

 - w: 쓰기 가능. 변경 사항을 저장할 수 있습니다.

 - x: 실행 가능. 프로그램 또는 스크립트 파일인 경우 실행할 수 있고 폴더인 경우 하위 폴더 및 파일에 액세스할 수 있음을 의미합니다.

 - -: 권한이 설정되지 않았습니다.
