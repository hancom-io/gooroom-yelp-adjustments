다른 애플리케이션으로 파일 열기

파일에서 파일을 두 번 클릭(또는 가운데 버튼 클릭)하면 해당 파일 유형에 대해 설정된 기본 애플리케이션으로 파일을 엽니다. 파일을 다른 애플리케이션에서 열거나 온라인에서 애플리케이션을 검색하거나 동일한 유형의 모든 파일에 대한 기본 애플리케이션 설정을 변경할 수 있습니다.

파일을 다른 애플리케이션으로 열려면 파일을 마우스 오른쪽 버튼으로 클릭한 다음 원하는 애플리케이션을 선택합니다. 원하는 애플리케이션이 표시되지 않으면 [다른 프로그램으로 열기]를 선택합니다. 
[파일 관리자]는 기본적으로 파일 처리 애플리케이션으로 알려진 항목만 표시합니다. 
시스템에 설치된 모든 애플리케이션을 목록을 보려면 [모든 프로그램 보기(V)]를 선택합니다.


기본 애플리케이션 변경
특정 파일 유형을 여는 데 사용되는 기본 애플리케이션을 변경할 수 있습니다. 이렇게 하면 원하는 애플리케이션으로 파일을 열 수 있습니다. 예를 들어 MP3 파일을 두 번 클릭하는 경우 특정 음악 플레이어가 실행되도록 할 수 있습니다.
          

1. 기본 애플리케이션을 변경하려는 파일 유형을 선택합니다. 예를 들어 MP3 파일을 여는 데 사용되는 애플리케이션을 변경하려면 .mp3 파일을 선택합니다.

2. 파일을 마우스 오른쪽 버튼으로 클릭하고 메뉴에서 [속성]을 선택합니다.

3. [열 프로그램] 탭을 선택합니다.

4. 원하는 애플리케이션을 선택하고 [기본으로 설정]을 클릭합니다.

해당 파일 유형에 대해 간혹 다른 프로그램을 사용하지만 기본 애플리케이션으로 설정하고 싶지는 않은 경우 해당 애플리케이션을 선택하고 [추가(A)]를 클릭하여 추천 프로그램에 추가합니다. 이렇게 하면 파일을 마우스 오른쪽 버튼으로 클릭한 다음 해당 애플리케이션으로 열기를 선택할 수 있습니다.
          

위 절차를 수행하면 선택한 파일 뿐만 아니라 동일한 유형의 모든 파일에 대한 기본 애플리케이션이 변경됩니다.
