파일 동작 환경 설정

한 번 클릭으로 파일을 실행할지 여부, 실행 가능한 텍스트 파일을 처리하는 방법 및 휴지통 비우기 동작 등을 설정합니다.

창의 오른쪽 상단에 있는 메뉴 버튼을 클릭하고 [기본 설정(P)]을 선택한 다음 [동작] 탭을 선택합니다.


열기 동작
한 번 클릭하여 파일 열기

기본적으로 한 번 클릭하면 파일이 선택되고 두 번 클릭하면 파일이 열립니다. 파일과 폴더를 한 번 클릭하여 열도록 설정할 수 있습니다. 한 번 클릭하여 열기로 설정하면 Ctrl을 누른 상태에서 클릭하여 여러 파일을 선택할 수 있습니다.




실행 가능한 텍스트 파일 동작
실행 가능한 텍스트 파일은 실행할 수 있는 프로그램이 포함된 파일입니다. 파일을 프로그램으로 실행하려면 파일 사용 권한을 허용해야 합니다. 이러한 파일 유형의 예는 Shell, Python 및 Perl 스크립트입니다. 각각 .sh , .py 및 .pl 확장자를 사용합니다.

실행 가능한 텍스트 파일에 대해 설정할 수 있는 옵션은 다음과 같습니다.

 - 실행 가능한 텍스트 파일 실행

 - 실행 가능한 텍스트 파일 보기

 - 매번 물어보기

[매번 물어보기]를 선택한 경우 선택한 텍스트 파일을 실행할지 묻는 대화 상자가 나타납니다.

실행 가능한 텍스트 파일을 스크립트라고도 합니다. ~/.local/share/nautilus/scripts 폴더의 모든 스크립트는 빠른 메뉴(Context Menu)에 나열됩니다.
로컬 폴더에서 스크립트를 실행하면 선택한 모든 파일이 매개 변수로 스크립트에 추가됩니다.
파일에서 스크립트를 실행하려면 다음을 수행합니다.
1. 원하는 폴더로 이동하여 파일을 선택합니다.

2. 파일을 마우스 오른쪽 버튼으로 클릭한 다음 빠른 메뉴에서 [열기]를 선택합니다.
