파일 및 폴더 삭제

파일 또는 폴더가 더 이상 필요하지 않은 경우 삭제할 수 있습니다. 삭제한 항목은 휴지통 폴더로 이동하고 휴지통을 비울 때까지 유지됩니다.

실수로 삭제한 경우 휴지통 폴더의 항목을 원래 위치로 복원할 수 있습니다.
 



파일을 휴지통으로 보내기
1. 휴지통으로 이동할 항목을 선택합니다.

2. Delete를 누릅거나 항목을 사이드 바의 휴지통으로 드래그합니다.

휴지통에서 [선택한 항목 복구]를 선택하면 파일이 원래 위치로 복원됩니다.

파일을 영구적으로 삭제하여 디스크 공간을 확보하려면 휴지통을 비워야 합니다. 휴지통을 비우려면 사이드 바에서 휴지통을 마우스 오른쪽 버튼으로 클릭한 다음 [휴지통 비우기]를 선택합니다.



영구적으로 파일 삭제
파일을 휴지통으로 보내지 않고 완전히 삭제할 수 있습니다.

파일을 완전히 삭제하기
1. 삭제할 항목을 선택합니다.

2. Shift를 길게 누른 상태에서 Delete를 누릅니다.

3. 파일 또는 폴더 삭제 확인 메시지가 표시됩니다. 이 작업은 취소할 수 없습니다.




이동식 장치에서 삭제한 파일은 Windows 또는 macOS와 같은 다른 운영 체제에서 보이지 않을 수 있습니다. 해당 파일 정보는 여전히 디스크에 있으며 장치를 컴퓨터에 다시 연결하면 사용할 수 있습니다.
