파일 찾기

생성하거나 다운로드한 파일을 찾기 힘든 경우 다음 팁을 참조합니다.

 - 파일을 저장한 위치를 기억하지 못하지만 이름을 알고 있는 경우 이름으로 파일을 검색합니다.

 - 파일을 방금 다운로드한 경우 웹 브라우저에서 파일을 공통 폴더에 자동으로 저장했을 수 있습니다. 홈 폴더에서 데스크톱 및 다운로드 폴더를 확인합니다.

 - 실수로 파일을 삭제했을 수 있습니다. 파일을 삭제하면 휴지통으로 이동하여 수동으로 휴지통을 비울 때까지 유지됩니다. 삭제된 파일을 복원하는 방법은 휴지통에서 파일 복원을 참조합니다.

 - 속성을 숨김 파일로 변경하는 과정에서 . 접두사가 추가되어 파일 이름이 변경되었을 수 있습니다. 접두사 . 또는 접미사 ~가 포함된 파일은 파일 관리자에서 표시되지 않습니다. [보기 옵션]을 클릭한 다음 [숨김 파일 표시]를 선택하여 숨김 파일을 표시합니다. 자세한 내용은 파일 숨기기를 참조합니다.
