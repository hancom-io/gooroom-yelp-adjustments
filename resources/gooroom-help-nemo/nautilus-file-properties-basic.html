파일 속성

파일 또는 폴더에 대한 정보를 보려면 마우스 오른쪽 버튼으로 클릭하고 [속성]을 선택합니다. 또는 파일을 선택하고 Alt+Enter를 누릅니다.

파일 속성 창에는 파일 유형, 파일 크기 및 마지막으로 수정한 시간 등의 정보가 표시됩니다. 이 정보를 자주 참조하는 경우 목록 보기 열 또는 아이콘 캡션에 표시하도록 설정할 수 있습니다.

기본 탭에서 제공하는 정보는 아래에 설명되어 있습니다. 이미지 및 비디오와 같은 특정 파일 유형의 경우 크기, 재생 시간 및 코덱과 같은 정보가 포함된 추가 탭을 제공합니다.


기본 속성
이름
이 필드를 변경하여 파일 이름을 바꿀 수 있습니다. 속성 외부에서 파일 이름을 바꿀 수도 있습니다. 이름 변경 방법은 파일 또는 폴더 이름 바꾸기를 참조합니다.

형식
PDF 문서, OpenDocument 텍스트 또는 JPEG 이미지와 같은 파일 유형을 식별하는 데 도움이 됩니다. 형식에 따라 파일을 여는 데 사용하는 애플리케이션이 달라집니다. 
예를 들어 음악 플레이어로는 사진을 열 수 없습니다. 자세한 내용은 다른 애플리케이션으로 파일 열기를 참조합니다.

파일의 MIME 유형은 괄호 안에 표시됩니다. MIME 유형은 컴퓨터가 파일 유형을 참조하는 데 사용하는 표준 방식입니다.

내용
이 필드는 폴더의 속성을 볼 때 표시됩니다. 폴더의 항목 수를 확인하는 데 도움이 됩니다. 폴더에 다른 폴더가 포함된 경우 각 하위 폴더는 추가 항목이 포함되어 있어도 하나의 항목으로 계산됩니다. 각 파일도 하나의 항목으로 계산됩니다. 폴더가 비어 있으면 이 필드에 아무것도 표시되지 않습니다.

크기
이 필드는 파일을 보고 있는 경우 표시됩니다. 파일 크기는 사용하는 디스크 공간의 크기를 나타냅니다. 
크기 정보는 파일을 다운로드하거나 이메일에 첨부하여 보내는 데 걸리는 시간을 예측하는 지표로 사용할 수도 있습니다(파일 크기가 큰 경우 보내거나 받는 데 더 많은 시간 필요).

크기는 Byte, KB, MB 또는 GB 단위로 표시될 수 있습니다. Byte 이외의 단위인 경우 괄호 안에 Byte 크기도 표시됩니다. 참고로 1KB는 1024Byte, 1MB는 ​​1024KB입니다.

상위 폴더
컴퓨터에서 각 파일의 위치는 절대 경로로 제공됩니다. 이는 파일의 고유한 "주소"이며 파일을 찾기 위해 이동해야 하는 폴더 목록으로 구성됩니다. 예를 들어 Jim의 Home 폴더에 Resume.pdf라는 파일이 있는 경우 상위 폴더는 /home/jim이고 위치는 /home/jim/Resume.pdf입니다.

빈 공간
폴더에 대해서만 표시됩니다. 폴더가 있는 디스크에서 사용할 수 있는 디스크 공간의 양을 제공합니다. 하드 디스크가 가득 찼는지 확인하는 데 유용합니다.

접근 시각
파일을 마지막으로 연 날짜 및 해당 시간입니다.

바뀐 시각
파일을 마지막으로 변경하고 저장한 날짜 및 해당 시간입니다.
