안전하게 외장 드라이브 제거

USB 플래시 드라이브와 같은 외부 저장 장치를 사용 중일 때 물리적으로 연결을 해제하면 파일이 손실 또는 손상될 수 있으므로 먼저 안전하게 제거하는 단계를 거쳐야 합니다. 
CD 또는 DVD 등 광 디스크를 사용하는 경우 이 단계를 수행하여 시스템에서 디스크를 꺼낼 수 있습니다.
 

이동식 장치 꺼내기
사이드 바의 장치 이름 옆에 있는 연결 해제 아이콘을 클릭하여 장치를 안전하게 제거하거나 꺼냅니다.

또는 사이드 바에서 장치 이름을 마우스 오른쪽 단추로 클릭한 다음 관련 메뉴를 선택합니다.




사용 중인 장치 안전하게 제거하기
장치에 저장된 파일이 열려 있고 다른 애플리케이션에서 사용 중인 경우 장치를 안전하게 제거할 수 없습니다. 볼륨이 사용 중이라는 메시지가 표시됩니다.

장치를 안전하게 제거하기
1. 장치의 모든 파일을 닫습니다.

2. 연결 해제 아이콘을 클릭하여 장치를 안전하게 제거하거나 꺼냅니다.
   또는 사이드 바에서 장치 이름을 마우스 오른쪽 단추로 클릭한 다음 관련 메뉴를 선택합니다.
