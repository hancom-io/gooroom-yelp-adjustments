사진 열기

이미지 보기에서는 다음과 같은 방법으로 사진을 열 수 있습니다. 


     1. 파일 보기 창에서 사진을 두 번 클릭하면 [이미지 보기]가 열립니다. [이미지 보기]가 기본 애플리케이션이 아닌 경우 기본 프로그램으로 설정할 수 있습니다.


     2. 사진을 두 번 클릭하면 기본적으로 각 사진마다 [이미지 보기] 창이 하나씩 열립니다. 선택에 따라 이미지 여러 개를 하나의 창에서 볼 수도 있습니다.


     3. 이미지 보기 창에서 직접 사진을 열 수도 있습니다.
1. [열기]를 클릭하거나 Ctrl + O 바로 가기 키를 사용하면 [이미지 보기] 창이 열립니다.

2. 사진을 선택하고 [열기] 버튼을 클릭합니다.
