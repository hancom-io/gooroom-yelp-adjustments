사진을 기본으로 이미지 보기에서 열기

     특정 이미지 파일 형식을 열 때 사용할 기본 애플리케이션을 이미지 보기로 변경하려면 다음을 수행합니다.

     
1. [파일]에서 파일을 마우스 오른쪽 단추로 클릭한 다음 [속성]을 선택합니다.

2. [다른 프로그램으로 열기] 탭으로 이동합니다. 추천 애플리케이션 목록이 나타납니다.

3. [이미지 보기]를 선택한 다음 [기본으로 설정]을 클릭합니다.

[이미지 보기]가 추천 애플리케이션 목록에 없으면 [다른 프로그램 보기]를 클릭합니다. [이미지 보기]를 선택하고 [기본으로 설정]을 클릭합니다.


4. [닫기]를 클릭합니다.
이제 해당 파일 형식을 두 번 클릭할 때마다 [이미지 보기]가 열립니다.
