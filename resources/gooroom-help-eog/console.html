명령 행
터미널에서 이미지 보기를 시작하려면 eog를 입력합니다.

명령 행
eog 명령 다음에 파일 이름을 입력하여 특정 파일을 열 수 있습니다.
eog image.jpeg

eog 명령 다음에 폴더 이름을 입력하여 특정 폴더를 열 수 있습니다.
eog folder

폴더의 모든 이미지를 보려면 이미지 모음을 탐색합니다.

전체 화면 모드에서 이미지 열기
eog --fullscreen image.jpeg


이미지 모음 기능 없이 이미지 열기
eog --disable-gallery image.jpeg


새 인스턴스에서 이미지 열기
eog --new-instance image.jpeg


슬라이드 쇼 모드에서 폴더 열기
eog --slide-show 사진/
