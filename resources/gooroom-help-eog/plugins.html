플러그인
eog-plugins 패키지에 이미지 보기의 추가 플러그인이 포함되어 있습니다. 플러그인을 사용하면 다음과 같은 확장 기능이 추가됩니다.

eog-plugins
Linux 배포판의 패키지 관리 프로그램에서 eog-plugins를 설치할 수 있습니다.
           
           sudo apt-get install eog-plugins
다음 기능이 eog-plugins에 포함되어 있습니다.

		 
- EXIF 표시: 디지털 사진의 카메라 설정 및 히스토그램을 표시합니다.

- 전체 화면 배경: 전체 화면 모드의 배경을 바꿀 수 있습니다.

- 두 번 눌러 전체 화면: 두 번 클릭하여 전체 화면 모드로 전환합니다.

- 지도: 지도에서 디지털 사진의 지리적 위치를 표시합니다.

- Picasa 웹 업로드: 사진을 Picasa 웹에 업로드합니다.

- Python 콘솔: 이미지 보기 Python 콘솔입니다.

- 메일로 보내기: 사진을 메일에 첨부해서 친구에게 보냅니다.

- 슬라이드 쇼 임의 모드: 사진을 슬라이드 쇼로 표시할 때 임의 순서로 표시합니다.

- 이미지 너비에 맞춰 확대/축소: 이미지 너비가 창 크기에 맞도록 확대/축소 수준을 조절합니다.
[기본 설정] 창의 [플러그인 탭]을 사용해 플러그인을 관리할 수 있습니다.
