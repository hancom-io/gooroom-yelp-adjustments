사진 편집 또는 삭제하기




사진 편집하기
이미지 보기는 사진을 보기 위해서만 사용할 수 있고 편집할 수는 없습니다. 사진을 편집하려는 경우 GIMP 이미지 편집기와 같은 사진 편집 애플리케이션을 사용해야 합니다. 다른 애플리케이션에서 편집하기 위해 사진을 열려면 다음을 수행합니다.

1. 헤더 바 메뉴의 [다른 프로그램으로 열기]를 선택하거나 이미지를 마우스 오른쪽 단추로 클릭합니다.

2. 사진을 편집하는 데 사용할 애플리케이션을 선택합니다.

3. 편집을 마치면 이미지를 저장하고 해당 애플리케이션을 닫습니다.

4. 이미지 보기에서 변경된 사진을 감지하여 다시 로드합니다.



사진 삭제하기 
더 이상 필요하지 않은 사진을 삭제하려면 마우스 오른쪽 단추를 클릭하고 [휴지통으로 옮기기]를 선택합니다.
