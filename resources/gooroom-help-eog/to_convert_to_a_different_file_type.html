다른 파일 형식으로 변환하기
사진은 JPEG 또는 PNG 등의 특정 파일 형식으로 저장됩니다. 필요에 따라 이러한 파일을 다른 형식으로 변환할 수 있습니다.



사진 하나를 다른 파일 형식으로 변환하기

1. [다른 이름으로 저장]을 클릭하면 저장을 위한 새 창이 열립니다.
           
2. [이름]에서 변환하려는 형식에 해당하는 파일 확장자를 지정합니다.

    예를 들어 JPEG(.jpg) 형식을 비트맵(.bmp) 형식으로 변환하려면 다음과 같이 지정합니다.
    
        원본 파일: image.jpg
        대상 파일: image.bmp
        
3. [저장]을 누르면 지정한 파일 형식으로 새롭게 저장합니다. 원본 파일은 유지됩니다.
이미지 보기는 선택한 파일 확장자에 따라 사진을 저장할 파일 형식을 인식합니다. 입력한 파일 확장자 또는 사용할 확장자를 인식하지 못하는 경우 [이미지 저장] 창에서 [지원하는 이미지 파일]을 클릭한 다음 드롭다운 상자에서 파일 형식을 선택할 수 있습니다.




여러 사진을 다른 파일 형식으로 변환하기

사진 여러 개를 다른 형식으로 한 번에 변환할 수 있습니다.
    1. Ctrl 키를 누른 채로 이미지 모음에서 변환하려는 이미지를 클릭합니다. 또는 Ctrl + A 키를 눌러 모든 이미지를 선택합니다.
    
    2. [다른 이름으로 저장] 대화 상자가 나타납니다.

       

    3. 변환하는 사진은 기본적으로 현재 폴더에 저장됩니다. 다른 위치에 저장하려면 [저장할 폴더]를 변경합니다.

    4. [파일 이름 형식] 오른쪽 드롭다운 목록에서 대상 파일 형식을 선택합니다.

    5. [다른 이름으로 저장]을 클릭합니다.
참고
여러 개의 사진 이름을 한 번에 바꾸기 - 여러 개의 사진 이름을 자동으로 바꾸거나 번호를 매깁니다.
