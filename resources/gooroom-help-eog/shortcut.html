키보드 바로 가기

다음은 이미지 보기에서 사용할 수 있는 키보드 바로 가기 키 전체 목록입니다.


열기, 닫기, 저장, 인쇄 
- 이미지 파일 열기: Ctrl + O
         
- 이미지를 같은 파일 이름으로 저장: Ctrl + S

- 현재 이미지의 복사본을 새 파일 이름으로 저장: Shift + Ctrl + S

- 현재 이미지 인쇄: Ctrl + P

- 현재 창을 닫기: Ctrl + W

- 현재 이미지를 바탕 화면 배경으로 사용: Ctrl + F8



이미지 보기 
- 폴더의 이전 이미지로 이동: 백스페이스 / ←
         
- 폴더의 다음 이미지로 이동: →

- 폴더의 첫번째 이미지로 이동폴더의 첫번째 이미지로 이동: Alt + Home

- 폴더의 마지막 이미지로 이동: Alt + End

- 폴더의 임의의 이미지로 이동: Ctrl + M

- 이미지 갤러리: Ctrl + F9

- 전체 화면: F11

- 슬라이드 쇼: F5

- 실제 크기: Ctrl + 0

- 창에 맞춤: F

- 큰 이미지 스크롤: Alt + 화살표

- 이미지 복사: Ctrl + C



확대/축소 및 회전 
- 확대: +
         
- 축소: -

- 시계 방향 회전: Ctrl + R

- 시계 반대 방향 회전: Shift + Ctrl + R

확대/축소


기타 
- 가장자리 창 F9

- 이미지 속성: Alt + Enter

- 실행 취소: Ctrl + Z

- 도움말: F1
