폴더의 전체 사진 보기

사진을 열고 [이전] 및 [다음] 버튼을 클릭하여 폴더의 모든 이미지를 빠르게 살펴볼 수 있습니다. 이미지 갤러리에서는 폴더의 모든 사진이 썸네일 미리 보기로 표시됩니다.



폴더에 포함된 사진 둘러보기

사진이 여러 개 포함된 폴더에서 개별 사진을 열지 않고도 편리하게 둘러볼 수 있습니다. 

1. 폴더의 사진을 엽니다.

2. 이전 또는 다음 버튼을 클릭하거나 왼쪽 또는 오른쪽 화살표를 누릅니다.

위 또는 아래 화살표를 누르거나 Space bar 또는 Backspace 키를 사용할 수도 있습니다.

3. 헤더 바에서 최대화 버튼을 눌러 사진을 전체 화면으로 표시하거나 F11 키를 눌러 최대화할 수 있습니다.

일반 모드로 돌아가려면 Esc 또는 F11 키를 누릅니다.


이미지 갤러리 보기

[이미지 갤러리]에서 폴더의 모든 사진을 미리 볼 수 있습니다. 사진이 많은 경우 빠르게 살펴보는 데 특히 유용합니다. 

1. 이미지 갤러리를 표시하려면 헤더 바 오른쪽의 버튼을 클릭하고 [이미지 갤러리] 확인란을 선택하거나 Ctrl + F9 바로 가기 키를 누릅니다.

2. 사진을 클릭하면 전체 크기로 표시됩니다.

3. 이미지 갤러리를 숨기려면 헤더 바 오른쪽의 버튼을 클릭하고 [이미지 갤러리] 확인란을 선택 해제하거나 Ctrl + F9 바로 가기 키를 누릅니다.
